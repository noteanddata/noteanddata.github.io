leetcode 139 Word Break

输入一个字符串和一个wordDict, 问这个字符串能不能通过这个wordDict组成

解题思路分析

  1. 这个题目基本的思路是动态规划(dynamic programming), 具体的解题思路分析在之前的笔记里面记录了
    http://www.noteanddata.com/leetcode-139-Word-Break-java-solution-note.html
  2. 当然， 前面分析的时候说bottom up的dp通常是prefix考虑的， postfix不太自然， 其实考虑postfix也可以很自然
    dp[i] |= dp[j] && wordDict.contains(s.substring(j, i)) // for all 0 <= j < i

java题解代码

public boolean wordBreak(String s, List<String> wordDict) {
    Set<String> set = new HashSet<>(wordDict);
    boolean[] dp = new boolean[s.length()+1];
    dp[0] = true;
    for(int i = 1; i <= s.length(); ++i) {
        for(int j = 0; j < i; ++j) {
            if(dp[j] && set.contains(s.substring(j, i))) {
                dp[i] = true;
            }
        }
    }
    return dp[s.length()];
}

时间复杂度

O(N^2)

空间复杂度

O(N)
