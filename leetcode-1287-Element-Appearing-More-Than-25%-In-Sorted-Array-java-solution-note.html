1287 Element Appearing More Than 25% In Sorted Array解题笔记

23 December 2019

题目 1287 Element Appearing More Than 25% In Sorted Array

  1. 输入一个非空的整数数组， 这个数组已经排序
  2. 其中有且只有一个数是出现超过25%的次数
  3. 求这个数

解题思路分析

  1. 这个题目感觉可能可以用一些巧妙的方法比如二分搜索做， 但是需要仔细分析。 简单一点的做法是， 因为知道只有一个数出现超过25%的次数， 那么， 直接求出现次数最多的数就好

java题解代码

public int findSpecialInteger(int[] arr) {
    int max = arr[0];
    int count = 1;
    int maxcount = 1;
    for(int i = 1; i < arr.length; ++i) {
        if(arr[i] == arr[i-1]) {
            count++;
            if(count > maxcount) {
                max = arr[i-1];
                maxcount = count;
            }    
        }
        else {
            if(count > maxcount) {
                max = arr[i-1];
                maxcount = count;
            }    
            count = 1;
        }
    }
    return max;
}

再分析一下， 因为数组不是空， 然后count只会在当前数和前面的数相等的情况下增加，所以取max只需要在这种情况下取就可以。
简化一点的代码, 如下。

public int findSpecialInteger(int[] arr) {
    int max = arr[0];
    int count = 1;
    int maxcount = 1;
    for(int i = 1; i < arr.length; ++i) {
        if(arr[i] == arr[i-1]) {
            count++;
            if(count > maxcount) {
                max = arr[i-1];
                maxcount = count;
            }    
        }
        else {
            count = 1;
        }
    }
    return max;
}

当然， 已经排序过的代码，其实是可以用二分搜索的方法来做， 这个下次再写。
