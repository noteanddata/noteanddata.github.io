主要笔记

  1. 之前的cost function的定义

cost(h(x), y) = - log(h(x)) // if y = 1
cost(h(x), y) = - log(1-h(x)) // if y = 0

因为y=0或者1， 所以这个函数可以进一步合并成
cost(h(x), y) = -y log(h(x)) - (1-y) log(1-h(x))

这样，就可以用梯度下降的方法求极值, 把训练数据做成向量表达以后，针对每个变量求偏导数，然后就可以梯度下降了
