题目 leetcode 1108 Defanging an IP Address

把一个ip地址里面的.换成[.]

解题思路分析

java非常简单， 直接一行就可以， 这里主要记录下学习 go的代码

java题解代码

public String defangIPaddr(String address) {
    return address.replaceAll("\\.", "[.]");
}

go语言题解代码1

func defangIPaddr(address string) string {
    var ret = make([]byte, 0)
    for _, ch := range address {
        if ch == '.' {
            ret = append(ret, '[')
            ret = append(ret, '.')
            ret = append(ret, ']')
        } else {
            ret = append(ret, byte(ch))
        }
    }
    return string(ret)
}

go语言题解代码2

import "strings"
func defangIPaddr(address string) string {
    var sb strings.Builder
    for _, ch := range address {
        if ch == '.' {
            sb.WriteString("[.]")
        } else {
            sb.WriteRune(ch)
        }
    }
    return sb.String()
}

go语言题解代码3

看了一下文档，才看到go语言里面也有直接Replace的一行函数

import "strings"
func defangIPaddr(address string) string {
    return strings.Replace(address, ".", "[.]", -1)
}
