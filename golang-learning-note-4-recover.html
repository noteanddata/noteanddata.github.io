基本概念

  1. recover可以获取之前panic的值， 然后停止异常向外抛出去的过程， 相当于java里面的catch.
    因为recover是用来catch panic的， 那么， 就是会在defer里面执行
  2. recover里面还可以修改函数的有名称的返回值
package main
import "fmt"

func main() {
	n := foo()
	fmt.Println("main received", n)
}

func foo() (m int) {
	defer func() {
		if err := recover(); err != nil {
			fmt.Println(err)
			m = 2
		}
	}()
	m = 1
	panic("foo: fail")
	m = 3
	return m
}

上面的代码的执行结果是

foo: fail
main received 2
