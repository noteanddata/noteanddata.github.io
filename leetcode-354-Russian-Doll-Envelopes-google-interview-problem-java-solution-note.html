leetcode 354 Russian Doll Envelopes解题笔记 (Google面试题)

15 March 2019

Google面试题 leetcode 354 Russian Doll Envelopes

  1. 输入一个数组， 每个元素也是一个数组， [width, height]， 表示一个信封的宽度和长度
  2. 如果一个a.width < b.width && a.height < b.height, 表示一个信封可以被完全放到另外一个信封里面
  3. 问，最多可以有多少个信封完全放到一起

解题思路分析

  1. 动态规划(dynamic programming)的特征比较明显
  2. 假设dp[j]已经是最大值， dp[i] = max(dp[j]+1) // if dp[j] can fit dp[i]
  3. 因为长宽都要满足， 所以，可以直接对输入数组进行排序, 然后就可以两层循环进行比较了。

java题解代码

public int maxEnvelopes(int[][] envelopes) {
    Arrays.sort(envelopes, new Comparator<int[]>() {
        public int compare(int[] a, int[] b) {
            return a[0] - b[0];
        }    
    });
    int[] dp = new int[envelopes.length];
    Arrays.fill(dp, 1);
    int max = 0;
    for(int i = 0 ; i < dp.length; ++i) {
        for(int j = 0; j < i; ++j) {                
            if(envelopes[j][0] < envelopes[i][0] && envelopes[j][1] < envelopes[i][1]) {
                dp[i] = Math.max(dp[i], dp[j]+1);
            }
        }
        max = Math.max(max, dp[i]);
    }
    return max;
}

时间复杂度

O(N^2)

空间复杂度

O(N)

思考

这个时间复杂度肯定不是最优的， 还有没有更好的做法？
如果只要长度或者宽度满足， 应该怎么做？
