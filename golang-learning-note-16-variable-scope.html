全局变量和局部变量的重名情况

package main

import "fmt"

var a int

func main() {
	var a int
	a = 4
	fmt.Println("a=", a)
	print()
}

func print() {
	fmt.Println("a=", a)
}

运行结果

a= 4
a= 0

结论1

外面有一个全局变量的情况下， 在函数里面可以有一个同名的局部变量， 定义了这个同名的局部变量以后， 那么在这个函数里面的修改，只是对这个局部变量做操作，对全局变量没有影响

修改上面的代码， 把main()函数里面的var a int 这行注释掉，

package main

import "fmt"

var a int

func main() {
	//	var a int
	a = 4
	fmt.Println("a=", a)
	print()
}

func print() {
	fmt.Println("a=", a)
}

运行的结果就是

a= 4
a= 4

局部变量用:=初始化的情况

局部变量用:=初始化的情况， 和声明一个变量效果是一样的

package main

import "fmt"

var a int

func main() {
	a := 4
	fmt.Println("a=", a)
	print()
}

func print() {
	fmt.Println("a=", a)
}

所以上面的代码的运行结果是

a= 4
a= 0
