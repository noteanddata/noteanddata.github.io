题目 leetcode 209 Minimum Size Subarray Sum

输入一个数组， 求一个最长的子数组， 使得这个子数组的和>=s

解题思路分析

  1. 这个题目的做法比较直接，用同向的双指针，分别向后移动，然后在每个窗口里面判断是否符合条件就可以

java题解代码

public int minSubArrayLen(int s, int[] nums) {        
    int sum = 0;
    int j = 0;
    int minlen = Integer.MAX_VALUE;
    for(int i = 0; i < nums.length; ++i) {
        while(sum < s && j < nums.length) {
            sum += nums[j];
            j++;
        }
        if(sum < s) {
            break;
        }
        int len = j-i;
        minlen = Math.min(minlen, len);
        sum -= nums[i];    
    }
    
    if(minlen == Integer.MAX_VALUE) {
        return 0;
    }
    else {
        return minlen;
    }
}

时间复杂度

O(N)

空间复杂度

O(1)
