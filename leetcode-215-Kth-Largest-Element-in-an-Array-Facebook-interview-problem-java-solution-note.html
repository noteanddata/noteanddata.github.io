Facebook面试题 Leetcode 215 Kth Largest Element in an Array

输入一个整数数组，返回其中第k大的元素

解题思路分析

  1. 把数组里面的每一个数都加入到一个priorityqueue里面，然后在加的过程中，如果priorityqueue的大小超过了k，就把最小的数去掉
  2. 最后，PriorityQueue的第一个元素就是第k大的元素

java题解代码

class Solution {
    public int findKthLargest(int[] nums, int k) {
        PriorityQueue<Integer> pq = new PriorityQueue<>(k);
        for(int n : nums) {
            pq.add(n);
            if(pq.size() > k) {
                pq.poll();
            }
        }
        return pq.poll();
    }
}

时间复杂度

O(N*lgK)

空间复杂度

O(K)
