leetcode 1021 Remove Outermost Parentheses解题笔记

06 April 2019

题目 leetcode 1021 Remove Outermost Parentheses

  1. 这个题目考虑的字符串里面只有左括号"(“和右括号”)"
  2. 定义一个字符串是valid的情况有三种
    a. 空字符串是valid的
    b. (s), 如果s是valid的
    c. a+b, 如果a和b是valid的
  3. 定义一个字符串是primitive的意思是这个字符串不能被分成a+b
  4. 输入一个字符串S，把这个字符串分解成S = P_1 + P_2 + … + P_k, 其中每个字符串P_i都是一个primitive的， 然后把每个P_i的最外层的左括号和右括号去掉， 然后求处理后的字符串

解题思路分析

  1. 一个字符串如果可以被分成a+b, 那么， a和b都是valid的， 那么， a就是自闭合的
  2. 如果P_i是primitive的， 那么， P_i就是自闭合的最小的完整字符串，所以，只要从左到右扫描，找到最小的完整的匹配的字符串，就是一个primitive
  3. 所以，每次计数一下当前没有被匹配的括号，遇到0的时候就处理一下就好

java题解代码

class Solution {
    public String removeOuterParentheses(String S) {
        int count = 0;
        int last = 0;
        StringBuilder sb = new StringBuilder();
        for(int i = 0; i < S.length(); ++i) {
            if(S.charAt(i) == '(') {
                count++;
            }
            else {
                count--;
                if(count == 0) {
                    sb.append(S.substring(last+1, i));
                    last = i + 1;
                }
            }
        }
        return sb.toString();
    }
}

时间复杂度

O(N)

空间复杂度

O(1)
