python学习笔记3 – set数据类型

set和其他语言里面的set基本类似

>>> x = {"apple", "banana", "cherry"}   
>>> print(x)
{'banana', 'cherry', 'apple'}
>>> print("apple" in x)
True
>>> print("blue" in x)
False
>>> print(len(x))
3
>>> x.add("orange")
>>> print(x)
{'banana', 'cherry', 'apple', 'orange'}
>>> x.remove("apple")
>>> print(x)
{'banana', 'cherry', 'orange'}
