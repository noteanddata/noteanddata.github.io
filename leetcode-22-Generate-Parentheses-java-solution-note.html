题目 leetcode 22 Generate Parentheses

  1. 输入一个n， 要求生成所有的有n个左括弧和右括弧的合法字符串

解题思路分析

  1. 这个题目可以用标准的backtrack来解， 对每一位，可以尝试左括号和右括号
  2. 参数里面需要传入剩下需要的leftcount 和rightcount
  3. 递归的开始要处理一些非法情况
  4. 如果遇到最终状态，那就加入到最终结果里面

java题解代码

public List<String> generateParenthesis(int n) {
    List<String> allList = new ArrayList<>();
    backtrack(n, n, "", allList);
    return allList;
}

public void backtrack(int leftcount, int rightcount, String cur, List<String> allList) {
    if(leftcount < 0 || rightcount < 0 || leftcount > rightcount) { // invalid 
        return ;
    }
    if(leftcount == 0 && rightcount == 0) {
        allList.add(cur);
        return;
    }
    backtrack(leftcount-1, rightcount, cur + "(", allList);
    backtrack(leftcount, rightcount-1, cur + ")", allList);
}
