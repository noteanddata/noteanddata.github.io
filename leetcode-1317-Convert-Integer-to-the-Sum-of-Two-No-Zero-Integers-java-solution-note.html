题目 leetcode 1317 Convert Integer to the Sum of Two No-Zero Integers

输入一个整数n, 要求输出两个数a, b, 使得a+b=n,
然后要求a和b都不包含字符0

解题思路分析

题目数据量比较小， 然后又是第一题， 直接用brute force做了； 然后判断是否包含0也是直接用转换成字符串的方式做了。
应该还有更好的做法。

java题解代码

public int[] getNoZeroIntegers(int n) {
    for(int i = 1; i < n; ++i) {
        int b = n - i;
        if(!has0(i) && !has0(b)) {
            return new int[]{i, b};
        }
    }
    return null;
}
boolean has0(int a) {
    String s = String.valueOf(a);
    return s.contains("0");
}

时间复杂度

O(N)
