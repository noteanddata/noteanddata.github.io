leetcode 961 N-Repeated Element in Size 2N Array解题笔记

04 July 2019

题目 leetcode 961 N-Repeated Element in Size 2N Array

输入一个整数数组， 数组长度是2*N, 其中里面有N+1个数， 也就是N个数是完全一样的， 然后有一个数数出现了N次

解题思路分析

朴素的做法就数遍历， 然后统计次数；
也可以比较一些其他做法, 可以比较连续的三个数找到重复， 然后对一个特殊情况做一下处理

java题解代码1

public int repeatedNTimes(int[] A) {
    Map<Integer, Integer> map = new HashMap<>();
    for(int i = 0; i < A.length; ++i) {
        int count = map.getOrDefault(A[i], 0)+1;
        if(count > 1) {
            return A[i];
        }
        map.put(A[i], count);
    }
    return -1;    
}

java题解代码2

因为其他所有的元素都是一样都， 所以取连续都三个数，就有可能找到这个数， 如果所有连续的三个数都没有都话，
只可能是这种情况
[1,2,3,1]

public int repeatedNTimes(int[] A) {
    for(int i = 0; i < A.length-2; ++i) {
        if(A[i] == A[i+1] || A[i] == A[i+2]) {
            return A[i];
        }
    }
    return A[A.length-1];
}

go题解代码

go语言笔记： 对于一个map， 即使key不存在， 也可以使用类似
countmap[mykey]的方式来取值， 这时候值是0

func repeatedNTimes(A []int) int {
    var countmap = make(map[int]int)
    for i := 0; i < len(A); i++ {
        var count = countmap[A[i]]
        if count > 0  {
            return A[i]
        }
        countmap[A[i]] = count+1
    }
    return -1
}
