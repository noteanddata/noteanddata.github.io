go语言依赖管理-glide

03 September 2019

在之前的文章中记录了go语言依赖管理的一些笔记 http://www.noteanddata.com/golang-learning-note-17-dependency-management.html,
但是有一些开源代码的依赖管理并没有升级到最新的go module, 比如今天下载一个开源代码就依赖了glide,
这里记录一下glide的笔记

  1. 安装
    brew install glide

  2. 使用
    在代码目录下面执行
    glide install
    就可以安装当前代码所有的依赖

  3. 配置文件
    依赖的配置写在glide.yaml里面
    可以手动添加， 也可以用命令添加
    glide get github.com/foo/bar
    glide get github.com/foo/bar#^1.2.3
