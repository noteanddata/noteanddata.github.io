Facebook面试题目 leetcode 215 Kth Largest Element in an Array

  1. 从一个整数数组里面，选择第k大的元素

解题思路分析

  1. 之前做了一个用PriorityQueue的笔记http://www.noteanddata.com/leetcode-215-Kth-Largest-Element-in-an-Array-Facebook-interview-problem-java-solution-note.html, 这里写一个用quick select的做法
  2. 这里做一个用quick select的做法， 具体思路非常标准，就是快速选择算法的实现，每次partition把数组分成两个部分, 然后看第k大的会在那边， 然后选择其中一边进行递归
  3. 具体关于快速排序和快速选择， 在前面的笔记里面有更详细的介绍http://www.noteanddata.com/classic-algorithm-quick-sort.html, http://www.noteanddata.com/classic-algorithm-quick-select.html

java题解代码

class Solution {
    public int findKthLargest(int[] nums, int k) {
        return quickselect(nums, 0, nums.length-1, k);
    }
    
    int quickselect(int[] nums, int from, int to, int k) {
        int index = partition(nums, from, to);
        if(to-index+1 == k) {
            return nums[index];
        }
        else if(to-index+1 > k) {
            return quickselect(nums, index+1, to, k);
        }
        else {
            return quickselect(nums, from, index-1, k-(to-index+1));
        }
    }
    int partition(int[] nums, int from, int to) {
        int pivot = nums[from];
        int i = from+1, j = to;
        
        while(i <= j) {
            while(i <= j && nums[i] <= pivot) {
                i++;
            }
            while(i <= j && nums[j] >= pivot) {
                j--;
            }
            if(i < j) {
                swap(nums, i, j);
            }
        }
        swap(nums, from, j);
        return j;
    }
    void swap(int[] nums, int i, int j) {
        int t = nums[i];
        nums[i] = nums[j];
        nums[j] = t;
    }
}

时间复杂度

O(N)

空间复杂度

O(1)
