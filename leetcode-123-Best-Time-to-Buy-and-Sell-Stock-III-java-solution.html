leetcode 123 Best Time to Buy and Sell Stock III 解题笔记

09 June 2019

题目 leetcode 123 Best Time to Buy and Sell Stock III

输入一个整数数组，里面每个元素表示某一天的股票价格，然后要求在最多做两次交易的情况下， 返回最大的利润。

刷题感悟

  1. 这个题目还有一个通用的dp做法， 下次再学习
  2. 同样， 还可以把遍历的次数从3次改成两次， 数组也可以只用一个数组， 后序再改进

解题思路分析

  1. 基本思路是从左到右扫一遍， 然后从右到左扫一遍, 然后把两个结果综合一下。 可惜这个题目之前刷过，思路也记得， 这次写的时候又写的一塌糊涂， 最近睡眠不足（又找借口了），脑子感觉很糊。

  2. 首先先看下只做一次交易的思路 http://www.noteanddata.com/leetcode-121-Best-Time-to-Buy-and-Sell-Stock-java-solution-note.html, 核心思路是在从左到右遍历的时候， 保存当前遇到过的最小值， 然后计算当前可能的最大利润。

  3. 然后我们想如果做两次交易怎么获得最大利润？ 如果两次，一定会有一个分割点， 然后左边做一次，右边做一次。
    如果对左边和右边单独做， 那么可以用同样的做法。 但是如果要对每个分割点硬做，就是要O(N^2)

  4. 但是， 如果只考虑左边的话，我们只需要一次遍历就可以的。 就像leetcode 121一样，在遍历的过程中，可以得到最大值。
    那么，剩下的问题就是， 是不是有类似的方法可以计算右边的值？
    我们从左到右扫描的时候， 我们是保存目前遇到的最小值， 然后通过当前值减去最小值，可以得到最大利润。
    类似的，我们可以从右向左扫描，保存目前遇到的最大值，然后通过最大值减去当前值，就是右边的最大利润。

  5. 但是要求两个交易的最大值，我们需要一个数组保存这个中间结果， 然后得到最大值

  6. 这个从左向右扫描和从右向左扫描的思路，和leetcode 238 Product of Array Except Self这个题目有那么一点类似
    http://www.noteanddata.com/leetcode-238-Product-of-Array-Except-Self-java-solution-note.html

java题解代码

public int maxProfit(int[] prices) {
    if(prices.length <= 1) return 0;
    
    int[] left = new int[prices.length];
    int min = prices[0];
    for(int i = 1; i < prices.length; ++i) {
        left[i] = Math.max(left[i-1], prices[i]-min);
        min = Math.min(min, prices[i]);
    }
    
    int[] right = new int[prices.length];
    int max = prices[prices.length-1];
    for(int i = prices.length-2; i >= 0; --i) {
        right[i] = Math.max(right[i+1], max - prices[i]);
        max = Math.max(max, prices[i]);
    }        
    
    int ret = 0;
    for(int i = 0; i < prices.length; ++i) {
        int transaction2 = (i == prices.length-1) ? 0 : right[i+1];
        ret = Math.max(ret, left[i] + transaction2);
    }
    return ret;
}

java题解代码2

把从左向右的两次循环合并成一次

public int maxProfit(int[] prices) {
    if(prices.length <= 1) return 0;
    
    int[] right = new int[prices.length];
    int max = prices[prices.length-1];
    for(int i = prices.length-2; i >= 0; --i) {
        right[i] = Math.max(right[i+1], max-prices[i]);
        max = Math.max(max, prices[i]);
    }
    
    int ret = 0;
    int min = prices[0];        
    int leftprofit = 0;
    for(int i = 0; i < prices.length; ++i) {
        int rightprofit = (i == prices.length-1) ? 0 : right[i+1];
        leftprofit = Math.max(leftprofit, prices[i] - min);
        ret = Math.max(ret, leftprofit + rightprofit);
        
        min = Math.min(min, prices[i]);
    }
    return ret;
}

    }
    
    int[] right = new int[prices.length];
    min = prices[prices.length-1];
    for(int i = prices.length-2; i >= 0; --i) {
        right[i] = Math.max(0, prices[i] - min);
        min = Math.min(min, prices[i]);
    }
    int max = 0;
    for(int i = 0; i < prices.length-1; ++i) {
        max = Math.max(max, left[i] + right[i+1]);
    }
    return max;
}
