题目 leetcode 1028 Convert to Base -2

把一个整数转化成一个用"-2"来做base的数

刷题感悟

这个也想了好一会儿， 还好想出来了。 从小数字样本出发， 从其他样本出发归纳分析是分析和构造的好办法

解题思路分析

  1. 首先回顾了一下普通的10进制转化成二进制是如何实现的, 主要过程就是
    a. 对2取余，
    b. 然后把余数去掉，得到一个差
    c. 把差除以2， 重复步骤a

  2. 思考了一下， 这里如果转化成-2的话， 从右边开始数，对于第0位和第2位，和base 2是一样的，
    对于第1位和第3位，原来做减法的地方要改成做加法， 因为这位数的值最后是负值。
    所以这个逻辑清楚了以后，代码就非常直接了。

java题解代码

public String baseNeg2(int N) {
    if(N == 0) return "0";
    StringBuilder sb = new StringBuilder();
    
    while(N > 0) {
        int r = N%2;
        sb.append(r);
        if(sb.length()%2 == 0) {
            N = (N+r)/2;
        }
        else {
            N = (N-r)/2;
        }
    }
    return sb.reverse().toString();
}
