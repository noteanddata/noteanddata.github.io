课程资料：
https://www.coursera.org/learn/machine-learning/lecture/6Nj1q/multiple-features

主要笔记：
1. 之前的linear regression非常简化， 只有一个变量。 比如只用房子面积来预测房子价格。
现在更进一步， 用很多变量来预测， 比如用房子面积，房子有几个房间，附近学校的分数等来预测房子价格
但是仍然使用线性关系， 比如这样
y = k0 + k1 * x1 + k2 * x2 + k3 * x3 + k4 * x4

那么，这个公式就可以用两个vector相乘来表示
