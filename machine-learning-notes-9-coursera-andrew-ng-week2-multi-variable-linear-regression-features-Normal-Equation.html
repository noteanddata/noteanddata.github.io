主要笔记：

这节主要讲了一个矩阵的normal equation方法, 就是通过矩阵的转账，以及矩阵的求逆，直接在数学上得到linear regression的最佳解,
具体方式是从二次函数出发，cost function的最低值是顶点，也就是导数是0的地方。
那么，针对多个变量也是分别针对每个变量求导，然后找等于0的值。
整个数学推导过程只能看其他辅助资料了：）

计算的结论是，矩阵求逆是O(N^3)的复杂度，所以对于feature个数小于1000的情况， 可以考虑用normal equation,
再往上就考虑还是用梯度下降。

然后, normal equation方法只是针对linear regression可以使用， 后面一些其他更复杂的方法，就不合适用了， 而梯度下降方法有更广泛的适用范围
