leetcode 976 Largest Perimeter Triangle解题笔记

11 January 2019

题目 leetcode 976 Largest Perimeter Triangle

输入一个整数数组， 问，从里面选三个数作为三角形的边， 可以选出的最大的三角形的周长是什么

解题思路分析

  1. 三角形的基本条件是两边之和大于第三边， 也就是说最小的两条边之和大于最大边就可以
  2. 所以如果对数组进行排序， 那么从最大的数a[n-1]开始选做最大的一条边， 如果a[n-2]+a[n-3] > a[n-1], 那么就可以组成一个三角形，并且这就是答案。 如果这个条件不符合， 那么不需要尝试以a[n-1]为最大边的其他组合了， 因为不可能存在另外两条边会大于a[n-1]了，
    直接向前取a[n-2]作为最大边开始计算就好

java题解代码

public int largestPerimeter(int[] a) {
    Arrays.sort(a);
    int max = 0;
    for(int i = a.length-1; i >= 0; i--) {
        if(i-2 >= 0 && a[i-2] + a[i-1] > a[i]) {
            return a[i-2]+a[i-1]+a[i];
        }
    }
    return 0;
}

时间复杂度

对数组进行了排序， 所以时间复杂度是O(N*lgN)

空间复杂度

没有使用额外的空间， 所以是O(1)
