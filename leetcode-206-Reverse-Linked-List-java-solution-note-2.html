题目 leetcode 206 Reverse Linked List

反转链表

解题思路分析

  1. 前面写了递归的代码,http://www.noteanddata.com/leetcode-206-Reverse-Linked-List-java-solution-note-1.html， 写错一次， 现在写循环的代码， 又写错， 真是悲催
  2. 设置3个指针, prev, cur, next， 然后每次向前移动三个指针； 主要是要注意循环移动的时候第一注意指针是如何覆盖的， 第二开始和结束的边界条件要注意

java题解代码

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        if(null == head || null == head.next) return head;
        
        ListNode prev = null;
        ListNode cur = head;
        ListNode next = head.next;
        while(next != null) {
            cur.next = prev;
            prev = cur;
            cur = next;
            next = next.next;
        }
        cur.next = prev;
        return cur;
    }
}

时间复杂度

O(N)

空间复杂度

O(1)

错误的代码

最后一个cur的时候没有把next指针更新一下?

Input: [1,2,3,4,5]
Output: [5]
Expected: [5,4,3,2,1]
