03 May 2019

题目 leetcode 157 Read N Characters Given Read4

现在要从文件里面读n个字符串，可以用的API只有read4
int read4(char[] buf);

解题思路分析

  1. 基本步骤就是可以分成几种情况，
    第一是读的过程中遇到文件结束了， 这个需要判断每次返回的字符个数是不是<4
    第一是可以完整的读入4个字符的， 总共可以读n/4次
    第二是最后多出来的几个字符， n%4

  2. 整体可以分成两步

java题解代码

/**
 * The read4 API is defined in the parent class Reader4.
 *     int read4(char[] buf);
 */
public class Solution extends Reader4 {
    /**
     * @param buf Destination buffer
     * @param n   Number of characters to read
     * @return    The number of actual characters read
     */
    public int read(char[] buf, int n) {
        
        int total = 0;
        char[] temp = new char[4];
        for(int i = 0; i < n/4; ++i) {
            int count = read4(temp);
            System.arraycopy(temp, 0, buf, total, count);
            total += count;
            if(count < 4) {
                return total;
            }
        }
        if(n%4 > 0) {
            int remain = n%4;
            int count = read4(temp);
            count = Math.min(count, remain);
            System.arraycopy(temp, 0, buf, total, count);
            total += count;
        }
        return total;
    }
}

java题解代码2

  1. 上面的代码其实比较冗余，代码结构不清晰， 整体状态可以分成下面两部分
    a. 文件是否结束, 这个只需要看返回值就可以
    b. 需要拷贝多少个字符到buf里面， 这个其实需要取Math.min(count, n-total)
    其中total是目前已经读过的字符数

这样，可以直接在一个循环里面把所有过程用一个统一的逻辑完成

/**
 * The read4 API is defined in the parent class Reader4.
 *     int read4(char[] buf);
 */
public class Solution extends Reader4 {
    /**
     * @param buf Destination buffer
     * @param n   Number of characters to read
     * @return    The number of actual characters read
     */
    public int read(char[] buf, int n) {
        
        int total = 0;
        char[] temp = new char[4];
        boolean eof = false;
        while(!eof && total < n) {
            int count = read4(temp);
            if(count < 4) {
                eof = true;
            }
            count = Math.min(count, n-total);
            System.arraycopy(temp, 0, buf, total, count);
            total += count;
        }
        
        return total;
    }
}
