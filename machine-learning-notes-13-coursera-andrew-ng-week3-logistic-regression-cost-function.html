主要笔记

  1. 在之前的sigmoid函数以后，选择cost function的定义如下

cost(h(x), y) = - log(h(x)) // if y = 1
cost(h(x), y) = - log(1-h(x)) // if y = 0

因为训练数据的y只有0和1， 然后选择了上面的两个对数函数定义cost function, 那么就有下面的特征：
y=1的时候， 如果预测值=1， 那么cost=0, 也就是完全符合， 如果预测值=0， cost就等于无穷大，也就是完全不符合；
y=0的时候，如果预测值=0， 那么cost=0， 也就是完全符合， 如果预测值=1， cost就等于无穷大，也就是完全不符合；

取负号是为了把[0,1]之间的值映射到正数范围内
