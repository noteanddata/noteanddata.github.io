python学习笔记6 – panda Series

panda库也是python处理数值的另外一个强大的库， 通常和numpy配合使用。

其中，最基础的数据结构是Series, 表示一个一维数组， 但是可以有index作为标签。
创建方式可以通过传入numpy的ndarray来创建

>>> import pandas as pd
>>> import numpy as np
>>> data = np.array([1,2,3])
>>> s = pd.Series(data)
>>> print(s)
0    1
1    2
2    3
dtype: int64
>>> s2 = pd.Series(data, index=["jack", "lucy", "john"])
>>> print(s2)
jack    1
lucy    2
john    3
dtype: int64
>>> data2 = {"jack":1, "lucy":2, "john":3}
>>> s3 = pd.Series(data2)
>>> print(s3)
jack    1
lucy    2
john    3
dtype: int64
