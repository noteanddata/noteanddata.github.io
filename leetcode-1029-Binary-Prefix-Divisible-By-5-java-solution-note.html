leetcode 1029 Binary Prefix Divisible By 5解题笔记

30 March 2019

题目leetcode 1029 Binary Prefix Divisible By 5

  1. 输入一个数组A， 里面每个数A[i]是0或者1
  2. 同时定义N[i]表示把A[0]到A[i]连起来用一个二进制表示的数
  3. 要求返回一个数组answer, 其中answer[i]表示N[i]能否被5整除

其中， 1 <= A.length <= 30000

刷题感悟

  1. 这个题目是第一题， 难度是easy， 而且我看大家都提交的飞快， 可是我花了10分钟才搞清楚应该怎么做， 也是醉。
    感觉这个余数的技巧在很多题目里面都有很多应用， 一定要熟练掌握。

解题思路分析

  1. 直接暴力转化二进制到十进制肯定不可以。 后面的二进制会很大就没有办法转化了
  2. 考虑N[i]到N[i+1]的变化
    N[i+1] = N[i] * 2 +a[i+1]
    然后余数有两个非常重要的规律， 在很多题目里面有应用。 （证明暂时略， 下次再写）
(a+b)%c = (a%c + b%c)%c
a*b%c = (a%c*b)%c

那么， 就有
N[i+1]%5 = (N[i] * 2 + a[i+1])%5 = (N[i] * 2 % 5 + a[i+1]%5)%5 = (N[i]%5 * 2 + a[i+1]%5)%

也就是 r = (r * 2 % 5 + v%5)%5

java题解代码

public List<Boolean> prefixesDivBy5(int[] A) {
    List<Boolean> list = new ArrayList<>();
    int r = 0;
    for(int v : A) {
        r = (r * 2 % 5 + v % 5)%5;
        list.add(r == 0);
    }
    return list;
}

时间复杂度

O(N)

空间复杂度

O(N)
