Amazon面试题目 leetcode 1026 Maximum Difference Between Node and Ancestor解题笔记

13 April 2019

Amazon面试题目 leetcode 1026 Maximum Difference Between Node and Ancestor

  1. 输入一个二叉树，问任意一个节点和他的祖先节点的最大的差是多少

解题思路分析

  1. 由于每个节点和祖先节点的最大的差，一定出现在当前节点和min(Ancestor)和max(Ancestor)之间
  2. 所以这个可以用dfs来做，每次递归的时候，把目前遇到的最小值和最大值传给子节点，就可以在一次遍历的时候找到需要求解的结果

java题解代码

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    private int maxdiff;
    public int maxAncestorDiff(TreeNode root) {
        if(null == root) return 0;
        
        maxdiff = 0;
        dfs(root, root.val, root.val);
        return maxdiff;
    }
    
    public void dfs(TreeNode node, int min, int max) {
        if(null == node) return;
        
        int diff1 = Math.abs(node.val-min);
        int diff2 = Math.abs(node.val-max);
        maxdiff = Math.max(maxdiff, diff1);
        maxdiff = Math.max(maxdiff, diff2);
        dfs(node.left, Math.min(min, node.val), Math.max(max, node.val));
        dfs(node.right, Math.min(min, node.val), Math.max(max, node.val));
    }
}

时间复杂度

O(N)

空间复杂度

O(lgN)
