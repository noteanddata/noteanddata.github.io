题目 leetcode 268 Missing Number

从[0,1….., n]的数组里面， 删除掉一个， 然后结果是乱序的， 求这个删除的数字

解题思路分析

  1. 可以用一个map或者boolean数组标记这个不存在的数， 也可以通过求和直接得到结果。 （当然排序也是可以的）

java题解代码1

设置一个boolean 数组标记是否存在

public int missingNumber(int[] nums) {
    boolean[] flag = new boolean[nums.length+1];
    for(int i = 0; i < nums.length; ++i) {
        flag[nums[i]] = true;
    }
    
    int index = -1;
    for(int i = 0; i < nums.length+1; ++i) {
        if(!flag[i]) {
            index = i;
            break;
        }
    }
    return index;
}

java题解代码2

因为期望结果是连续的，所以和是可以提前知道的， 所以missing的数就可以用和来找到

public int missingNumber(int[] nums) {
    int sum = 0;
    for(int num: nums) {
        sum += num;
    }
    int total = (1 + nums.length) * nums.length / 2;
    return total - sum;
}

go语言题解代码1

func missingNumber(nums []int) int {
    existmap := make(map[int]bool, 0)
    for _, v := range nums {
        existmap[v] = true
    }
    for i := 0; i <= len(nums); i++ {
        if !existmap[i] {
            return i
        } 
    }
    return -1
}

go语言题解代码2

func missingNumber(nums []int) int {
    sum := 0
    for _, v := range nums {
        sum += v
    }
    total := (1+len(nums)) * len(nums) / 2
    return total-sum
}

go语言题解代码4

import "sort"
func missingNumber(nums []int) int {
    sort.Ints(nums)
    for i := 0; i < len(nums); i++ {
        if i != nums[i] {
            return i
        } 
    }
    return len(nums)
}
