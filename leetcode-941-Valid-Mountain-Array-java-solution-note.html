题目 Leetcode 941 Valid Mountain Array

判断一个数组是否是mountain array, 也就是一个先升序，后降序的数组

A is a mountain array if and only if:

    1. A.length >= 3
    2. There exists some i with 0 < i < A.length - 1 such that:
        A[0] < A[1] < ... A[i-1] < A[i]

解题思路分析

简单实现， 设置状态up， 分null, true, false三种， 依次判断是否从null->true->false,
代码不是很简洁，但是比较直观

java代码

public boolean validMountainArray(int[] A) {
        if(null == A || A.length < 3) return false;
        Boolean up = null;
        for(int i = 1; i < A.length; ++i) {
            if(A[i] == A[i-1]) {
                return false;
            }
            if(null == up) {
                if(A[i] > A[i-1]) {
                    up = true;
                }
                else {
                    return false;
                }
            }
            else if(up) {
                if(A[i] < A[i-1]) {
                    up = false;
                }
            }
            else {
                if(A[i] > A[i-1]) {
                    return false;
                }
            }
        }
        return !up;
}

时间复杂度

数组只遍历一次， O(N)

空间复杂度

只使用了一个常量, O(1)
