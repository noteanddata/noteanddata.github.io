go语言学习笔记27 – array和slice笔记1

  1. array的定义
    array就是一个固定大小的数组， 下面的方式可以定义一个array
var a [10]int

也可以这样

primes := [6]int{2, 3, 5, 7, 11, 13}

也可以让编译器来计算大小

b := [...]string{"Penn", "Teller"}
  1. 什么是slice
    slice是指“dynamically-sized, flexible view into the elements of an array”，
    这里有两个特点，
    a. slice是动态大小
    b. slice是数组的一个视图,

下面的方式可以定义一个slice, 先定义一个固定大小的数组以后，
再通过a[low : high]从中取一个slice

primes := [6]int{2, 3, 5, 7, 11, 13}
var s []int = primes[1:4]
  1. slice为什么是视图？
    因为slice本身不存储数据，内部是一个指向数组的指针, 在go语言里面的定义如下

https://github.com/golang/go/blob/6ffca2260296d82f235d38367f0d0f008121521e/src/runtime/slice.go#L13

type slice struct {
	array unsafe.Pointer
	len   int
	cap   int
}

所以修改slice里面的数据，就是修改指向的数组里面的数据
