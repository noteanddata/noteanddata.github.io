coursera的吴恩达机器学习课程第二周—多变量linear regression梯度下降时候的feature scaling

课程资料：
https://www.coursera.org/learn/machine-learning/lecture/xx3Da/gradient-descent-in-practice-i-feature-scaling

主要笔记：
1. 针对多变量的linear regression,
y = k0 + k1 * x1 + k2 * x2 + k3 * x3 + k4 * x4

如果直接使用梯度下降，可能会因为多个变量的值差距太大，导致收敛的速度太慢，各个等高线之间来回跳的时候速度不均衡？
所以建议把每个变量scale到近似的范围， 比如都scale到[-1,1]， 梯度下降的时候就会比较快
处理方法有
- 按照最大值来scale, 比如x[i] = x[i]/max(x), 这个方法的问题是虽然值最后都是[-1,1], 但是可能值有可能会变得很小
- 按照平均值来scale, 比如x[i] = (x[i] - avg(x)) / range[x]
其中range[x] = max(x) - min(x)
