18 November 2018

题目 Leetcode 944 Delete Columns to Make Sorted

  1. 输入一个字符串数组，其中每个字符串都是相同的长度
  2. 定义在index=k上的删除操作是指把数组中每个字符串在k这个位置的字符删除
  3. 求需要删除的index的最小个数， 使得完成删除操作以后，整个数组的字符串在剩下的每一个index上都是non-decreasing sorted order.

解题思路分析

  1. 这个题目也比较直接，就是看现在的字符串在每个index上有多少个是不符合排序规则的。
  2. 因为整个数组都要求在某一个index上排序，所以每次只需要和前一个比较就好

java代码

public int minDeletionSize(String[] A) {
       int count = 0;
       for(int i = 0; i < A[0].length(); ++i) {
           for(int j = 1;j < A.length; ++j) {
               if(A[j].charAt(i) < A[j-1].charAt(i)) {
                   count++;
                   break;
               }
           }
       }
       return count;
}

时间复杂度

两层循环，O(N*M), 其中N是数组的长度， M是数组每个字符串的长度

空间复杂度

只使用了一个常量， O(1)
