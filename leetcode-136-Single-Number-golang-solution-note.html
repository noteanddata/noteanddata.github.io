题目 leetcode 136 Single Number

输入一个数组， 只有一个数只出现一次， 其他数都出现两次， 要求找到这个一次的数

解题思路分析

  1. 通过异或的方式可以得到唯一的那个不同的值

go语言题解代码1

  1. go语言的位操作， 基本和其他语言类似
    https://www.tutorialspoint.com/go/go_bitwise_operators.htm
  2. 但是没有无符号右移， 测试了一下下面代码的结果还是-1， 也就是右移操作是带符号的
-1 >> 1

func singleNumber(nums []int) int {
    ret := 0
    for _, v := range nums {
        ret = ret ^ v
    }
    return ret
}

go语言题解代码2

在go语言的遍历过程中，range可以用来遍历数组和map

func singleNumber(nums []int) int {
    var countmap = make(map[int]int)
    for _, v := range nums {
        countmap[v]++
    }
    for k, v := range countmap {
        if v == 1 {
            return k
        }
    }
    return -1
}

go语言题解代码3

当然， 也可以排序做, sort.Ints排序以后， 这个数组的元素就发生了变化

import "sort"
func singleNumber(nums []int) int {
    sort.Ints(nums)
    for i := 0; i < len(nums) - 1; i += 2 {
        if nums[i] != nums[i+1] {
            return nums[i]
        }
    }
    return nums[len(nums)-1]
}
