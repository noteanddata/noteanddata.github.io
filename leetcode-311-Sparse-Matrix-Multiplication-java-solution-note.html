题目 leetcode 311 Sparse Matrix Multiplication

两个稀疏矩阵相乘

解题思路分析

  1. 暴力解就是直接按照矩阵乘法的公式计算每个值， 总共三层循环
  2. 由于三层循环没有任何条件判断， 那么循环可以换一下， 这样， 就可以对一些是0的地方跳过

java题解代码

public int[][] multiply(int[][] A, int[][] B) {
    
    int[][] ret = new int[A.length][B[0].length];
    for(int i = 0; i < ret.length; ++i) {
        for(int k = 0; k < A[i].length; ++k) {
            if(A[i][k] == 0) continue;
            for(int j = 0; j < ret[i].length; ++j) {                
                if(B[k][j] == 0) continue;
                ret[i][j] += A[i][k] * B[k][j];
            }
        }
    }
    return ret;
}

暴力解代码

public int[][] multiply(int[][] A, int[][] B) {
    
    int[][] ret = new int[A.length][B[0].length];
    for(int i = 0; i < ret.length; ++i) {
        for(int j = 0; j < ret[i].length; ++j) {                
            for(int k = 0; k < A[i].length; ++k) {
                ret[i][j] += A[i][k] * B[k][j];
            }
        }
    }
    return ret;
}
