主要笔记：
1. 针对多变量的linear regression,
y = k0 + k1 * x1 + k2 * x2 + k3 * x3 + k4 * x4

仍然使用类似的梯度下降方法，在一个学习速率(learning rate)下，同时分别针对多个变量求导数，然后找出方向下降的方向，然后逐步移动。

之前说了梯度下降这个过程需要是一个凸函数，才可以找到全局最低点。 而对于一个变量的linear regression来说，整个曲面是一个凸函数。
对于多变量，是否是凸函数，这节里面还没有讲到
