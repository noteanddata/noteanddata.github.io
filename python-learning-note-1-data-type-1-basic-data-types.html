python学习笔记1–基础数据类型

感觉要实践机器学习必须要稍微系统学习一下python基础知识，否则处理数据的时候不停的在查函数
这里先学习一下python的基础数据类型

字符串

  • 代码
s = "hello,world"
print(s)
  • 输出结果
hello,world

数值类型， 有int, float 和complex

complex就是数学上的复数

  • 代码
n1 = 3
f1 = 3.3
c1 = complex(5,5)

print(n1)
print(f1)
print(c1)
  • 输出
3
3.3
(5+5j)

boolean 类型

  • 代码
b1 = True
print(b1)
  • 输出
True
