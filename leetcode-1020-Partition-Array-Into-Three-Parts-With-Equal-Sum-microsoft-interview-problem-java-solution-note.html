微软面试题 leetcode 1020 Partition Array Into Three Parts With Equal Sum

输入一个整数数组，返回这个数组能否被分成三份，使得每部分的和相同

解题思路分析

从前到后遍历计算和(partsum)， 遇到等于sum/3就重置partsum, 如果可以找到两个这样的，就返回true

java题解代码

public boolean canThreePartsEqualSum(int[] A) {
    int sum = Arrays.stream(A).sum();
    if(sum%3 != 0) return false;
    int partsum = 0;
    int count = 0;
    for(int i = 0; i < A.length; ++i) {
        partsum += A[i];
        if(partsum == sum/3) {
            partsum = 0;
            count++;
            if(count == 2) {
                return true;
            }
        }
    }
    return false;
}

时间复杂度

O(N)

空间复杂度

O(1)

从头尾分别循环的代码

class Solution {
    public boolean canThreePartsEqualSum(int[] A) {
        int sum = 0;
        for(int i = 0; i < A.length; ++i) {
            sum += A[i];
        }
        if(sum%3 != 0) return false;
        
        int index1 = -1;
        int sum1 = 0;
        for(int i = 0; i < A.length; ++i) {
            sum1 += A[i];
            if(sum1 == sum/3) {
                index1 = i;
                break;
            }
        }
        if(index1 == -1) {
            return false;
        }
        int index2 = -1;
        int sum2 = 0;
        for(int i = A.length-1; i >= 0; i--) {
            sum2 += A[i];
            if(sum2 == sum/3) {
                index2 = i;
                break;
            }
        }
        if(index2 == -1) {
            return false;
        }
        return index2 > index1;
    }
}
