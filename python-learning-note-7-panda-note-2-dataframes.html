python学习笔记7 – panda DataFrames

panda里面的DataFrames是一个二维矩阵， 行和列都可以有index或者是label，有很多内置的各种矩阵操作，以及对外的接口比如读取csv等，

>>> import pandas as pd
>>> d2 = {"name": ["jack", "lucy"], "age": [22,21]}
>>> df2 = pd.DataFrame(data=d2)
>>> print(df2)
   name  age
0  jack   22
1  lucy   21

>>> df3 = pd.DataFrame(data=d3, index=["2020-01-01", "2020-01-02", "2020-01-03"])
>>> print(df3)
            temp  cost
2020-01-01    37     5
2020-01-02    33     3
2020-01-03    36     6


>>> print(df3.index)
Index(['2020-01-01', '2020-01-02', '2020-01-03'], dtype='object')
>>> print(df3.columns)
Index(['temp', 'cost'], dtype='object')
>>> print(df3.dtypes)
temp    int64
cost    int64
dtype: object

>>> df3.iloc[0]
temp    37
cost     5
Name: 2020-01-01, dtype: int64
