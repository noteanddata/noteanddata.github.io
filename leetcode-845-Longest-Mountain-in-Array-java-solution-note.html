题目 leetcode 845 Longest Mountain in Array解题笔记

28 April 2019

题目 leetcode 845 Longest Mountain in Array

输入一个整数数组，如果存在一个长度至少是3的子数组是先上升，后下降的话, 那么这个子数组就是mountain数组
就是子数组是 B[0] < B[1] < … B[i-1] < B[i] > B[i+1] > … > B[B.length - 1]

求最长的mountain数组的长度

解题思路分析

  1. 这类题目的核心是状态分析， 总共有几种情况:
    a. 如果是上升的第一个， 那么需要记录上升的开始index
    b. 如果是继续上升， 那么保持当前状态不变，开始index也不变
    c. 如果两个数相等， 那么所有状态reset， 进入非法状态
    d. 如果是下降的， 那么， 如果之前有合法的上升index， 那么就找到了一个mountain, 记录长度
    e. 凡事不是上升的, 就记录当前状态为一个状态， 下次遇到上升就是第一个上升。

java题解代码

public int longestMountain(int[] A) {
    int maxlen = 0;
    int upstart = -1;
    boolean firstup = true;
    for(int i = 1; i < A.length; ++i) {
        if(A[i] > A[i-1]) {
            if(firstup) {
                upstart = i-1;
                firstup = false;
            }
        }
        else if(A[i] == A[i-1]) {
            upstart = -1;
            firstup = true;
        }
        else { // A[i-1] > A[i] 
            if(upstart >= 0) {
                maxlen = Math.max(maxlen, i-upstart+1);
            }
            firstup = true;
        }
    }
    return maxlen;
}

时间复杂度

O(N)

空间复杂度

O(1)
