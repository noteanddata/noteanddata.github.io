题目 Facebook面试题–Leetcode 98 Validate Binary Search Tree

  1. 输入一个二叉树， 验证是否是一个二叉搜索树
  2. 二叉搜索树的定义
    a. 左边的所有节点都小于根节点
    b. 右边的所有节点都大于根节点
    c. 所有的子树都是二叉搜索树

解题思路分析

  1. 二叉搜索树的特点非常明显， 他的主要作用是方便搜索， 可以在O(lgN)的时间复杂度里面做搜索，
    作为验证，可以把这个树做一个in order遍历， 那么， in order的结果就是按顺序从小到大的。
  2. 如果使用额外的空间，可以把整个树序列化到一个数组里面。当然验证一个数组是否排序，只需要比较两个相邻的元素就好。
    所以我们可以只使用一个prev保存前一个元素就好

java题解报告

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    private Integer prev;
    private Integer cur;

    public boolean isValidBST(TreeNode root) {
        return dfs(root);
    }
    
    boolean dfs(TreeNode root) {
        if(null == root) return true;
        
        if(!dfs(root.left)) return false;
        if(null != prev && root.val <= prev) return false;
        
        prev = root.val;
        
        return dfs(root.right);
    }
}

时间复杂度

每个节点只遍历一次， O(N)

空间复杂度

O(1)
