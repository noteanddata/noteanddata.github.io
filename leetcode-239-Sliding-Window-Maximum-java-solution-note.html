leetcode 239 Sliding Window Maximum解题报告

06 May 2019

题目 leetcode 239 Sliding Window Maximum

  1. 输入一个整数数组和一个k， k表示一个sliding windows的大小
  2. 要求返回从左到右所有sliding windows大小是k的窗口大小

刷题感悟

  1. 这个题目用heap的做法并不是太难，不过最近状态非常差，这题目错了好几次才通过，真是郁闷

解题思路分析

  1. 可以维护一个大小为k的最大堆(heap)，每次遇到一个元素就把这个元素加到堆里面去， 如果堆的大小超过了k，就把最前面的元素剔除, 这样，这个堆的范围就是sliding windows的范围， 每次都可以取堆里面最大的元素就是sliding windows的最大元素

java题解代码

public int[] maxSlidingWindow(int[] nums, int k) {
    if(nums.length < k) return new int[]{};
    if(k == 0) return new int[]{};
    PriorityQueue<Integer> pq = new PriorityQueue<>(Comparator.reverseOrder());
    int[] ret = new int[nums.length-k+1];
    int j = 0;
    for(int i = 0; i < nums.length; ++i) {
        pq.add(nums[i]);
        if(pq.size() > k) {
            pq.remove(nums[i-k]);
        }
        if(pq.size() == k) {
            ret[j++] = pq.peek();
        }
    }
    return ret;
}

时间复杂度

O(N*lgK)

空间复杂度

O(K)
