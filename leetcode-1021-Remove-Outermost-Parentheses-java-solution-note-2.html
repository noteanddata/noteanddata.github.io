Google面试题 leetcode 1021 Remove Outermost Parentheses

  1. 题目分析和之前的双指针的做法参考这里 http://www.noteanddata.com/leetcode-1021-Remove-Outermost-Parentheses-java-solution-note.html

  2. 这里是学习了其他人的一些做法后，发现双指针还是略有繁琐，其实可以直接判断当前open的左括号的个数来判断当前字符是否需要被加入， 左括号是0的时候就表示最外层，就不需要加入。

  3. 判断的时候稍微要注意的是，左括号需要线判断是否加入再计数， 右括号需要先更新计数再判断是否加入

java题解代码

public String removeOuterParentheses(String S) {
    StringBuilder sb = new StringBuilder();
    int open = 0;
    for(int i = 0; i < S.length(); ++i) {
        if(S.charAt(i) == '(') {
            if(open > 0) {
                sb.append('(');
            }
            open++;
        }
        else {
            open--;
            if(open > 0) {
                sb.append(')');
            }
        }
    }
    return sb.toString();
}

时间复杂度

O(N)

空间复杂度

O(1)
