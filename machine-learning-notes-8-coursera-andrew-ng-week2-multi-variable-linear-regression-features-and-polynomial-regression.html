coursera的吴恩达机器学习课程第二周—feature的选择和非线性的regression

课程资料：
https://www.coursera.org/learn/machine-learning/lecture/Rqgfz/features-and-polynomial-regression

主要笔记：

  1. feature的选择， 比如以房价为例子， 可以选择房子的长度和宽度分别作为feature， 这样有两个变量，
    也可以选择房子的面积作为feature， 这样只有一个变量。
    因为机器学习涉及到大量的矩阵计算，所以feature多少会影响到速度
  2. 非线性的regression (多项式)， 之前的例子假设都是线性函数， 这样会有很大的限制，实际的变量关系很可能是非线性的。 比如
    y = k0 + k1 * x1 + k2 * x1^2
    但是，可以通过定义一个新的变量x2 = x1 ^2 , 这样，就变成了有两个变量的线性关系，仍然可以用linear regression来解。
    那么，这个linear regression可以应用的范围就一下子大了很多很多.
    当然，在这样的情况下， feature scaling就变得非常重要。 因为每个变量的大小就多了很多很多。 所以需要scaling到一个

当然，整个过程非常的科学，从最简单的一个变量的线性函数开始，然后过渡到多个变量的线性函数， 最后扩展到支持多个变量的多项式函数，一下子可以应用到很多很多场景了。
