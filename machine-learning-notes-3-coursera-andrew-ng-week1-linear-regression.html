coursera的吴恩达机器学习课程第一周-linear regression(线性回归)笔记

课程视频
https://www.coursera.org/learn/machine-learning/lecture/db3jS/model-representation

主要内容， 还是以房子价格和面积的关系为例子, 做有监督学习.
输入一堆面积和成交价， 要求得出一个预测函数，可以对以后的面积输出成交价。
这里只有一个变量和一个输出，预测函数从最简单的一次函数开始, y = kx + b
然后对一堆输入，做出一个尽可能好的预测函数，能够预测其他输入数据的价格.

这个方法就叫做线性回归 (linear regression), 因为这里首先用了一个线性函数，然后这是一个回归问题， 输出是一个连续的值而不是一个分类问题
