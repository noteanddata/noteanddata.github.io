javascript里面的var和let以及const的区别

  1. 看现代的很多javascript代码，会看到大量的const和let， 而很少有var, 出于好奇， 查看了一下var和let以及const的区别

  2. 在ECMAScript 2015标准(又称为在ECMAScript6, 简称ES6)里面， 给javascript引入了const和let两个关键字. 使用const和let声明的变量， 有block scope, 而之前的变量，只有global scope 和function scope两种。

  • 所谓global scope就是变量声明在函数外面， 那么就是全局变量
  • 所谓function scope就是变量声明在函数内部，就算在一个for循环里面，变量也是有函数作用域的， 这时候会造成很多的困扰
  • 所谓block scope, 就是块作用域， 基本和其他c系列的编程语言一样(c, c++, c#, java, golang等), 变量作用域到对应的大括弧结束
  1. 测试代码1
    把下面的代码保存成node1.js, 然后运行node node1.js
test1()
test2()

function test1() {
  console.log("-----in test1----")
  for(var i = 0; i < 3; ++i) {
    console.log("i=", i)
  }
  console.log(i) // will print 3 
}

function test2() {
  console.log("-----in test2----")
  for(let i = 0; i < 3; ++i) {
    console.log("i=", i)
  }
  console.log(i) // ReferenceError: i is not defined
}

    注释里面写了会打印的值

function test3() {
  var a = 3
  {
    let a = 4
    console.log("a=", a) // will print 4
  }
  console.log("a=", a) // will print 3
}

function test4() {
  var a = 3
  {
    var a = 4
    console.log("a=", a) // will print 4
  }
  console.log("a=", a) // will print 4
}
