09 June 2019

题目 leetcode 50 Pow(x, n)

自己实现幂函数pow(x, n)

解题思路分析

  1. 这个题目和前面的求平方根的数学题类似， 基本还是分治的思想， 前面是用二分求平方根， 这里是先求pow(x, n/2),
    然后再根据情况做处理
  2. 但是这里有一些细节是需要处理的，
    – 比如n是奇数和偶数
    – 比如n是负数
    – 比如递归什么时候结束

java题解代码

public double myPow(double x, int n) {
    if(n == 0) return 1.0;
    double half = myPow(x, n/2);
    if(n%2 == 0) {
        return half * half;
    }

    if(n > 0) {
        return half * half * x;
    }
    else {
        return  half * half / x;
    }
}

时间复杂度

O(lgN) ?
