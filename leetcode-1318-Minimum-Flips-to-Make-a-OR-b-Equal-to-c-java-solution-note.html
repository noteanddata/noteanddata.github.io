题目 leetcode 1318. Minimum Flips to Make a OR b Equal to c

输入3个整数a,b,c, 问， 对a和b需要最少把多少位flip一下， 使得a or b = c

解题思路分析

对每一位进行处理，
* 如果c这一位是1， 那么在a和b这两个数的这一位都是0的情况下才需要flip一位
* 如果c这一位是0， 那么在a和b任何一个数数的这一位如果是1的情况下都需要flip一位， 如果两个数都是1的话那就要flip两位

java题解代码

public int minFlips(int a, int b, int c) {
    int count = 0;
    for(int i = 0; i < 32; ++i) {
        int c0 = c & 1;
        int b0 = b & 1;
        int a0 = a & 1;
        
        if (c0 == 1) {
            if(a0 == 0 && b0 == 0) {
                count++;
            }
        } else { // c0 == 0
            count += a0 + b0;
        }
        
        c = c >> 1;
        b = b >> 1;
        a = a >> 1;
    }    
    return count;
}
