Amazon面试题题目 leetcode 252 Meeting Rooms

输入一个二维数组，里面每个元素表示一个会议的开始时间和结束时间， 问一个人能否参加全部的会议

解题思路分析

  1. 就是判断interval是否有冲突, 那么可以对interval按starttime先进排序， 然后判断两个相邻的是否有冲突就好

java题解代码

public boolean canAttendMeetings(int[][] intervals) {
    Arrays.sort(intervals, new Comparator<int[]>() {
        public int compare(int[] a, int[] b) {
            if(a[0] != b[0]) return a[0] - b[0];
            return a[1] - b[1];
        }
    });
    for(int i = 1; i < intervals.length; ++i) {
        if(intervals[i][0] < intervals[i-1][1]) {
            return false;
        }
    }    
    return true;
}

时间复杂度

O(N*lgN)

空间复杂度

O(1)
