ui设计学习笔记3 - 排版基础

看了一下一个零基础平面设计基础视频， 主要笔记如下：

  1. 相关 - 接近的内容尽量排版到一起， 在视觉上强调内容互相有关联
  2. 留白 - 平面设计中需要有留白， 这样内容就有呼吸的空间
  3. 对齐 - 相关内容需要对齐到一起
  4. 对比 - 制造对比差异大的视觉元素，突出重点
  5. 层次 - 建立层次感可以有重点的突出
  6. 重复 - 重复可以造成强化的效果
