题目 leetcode 905 Sort Array By Parity

输入一个整数数组， 返回一个新的数组， 数组的左边都是偶数，数组的右边都是奇数

解题思路分析

思路非常直接， 遍历数组， 如果是偶数就放在结果的左边，如果是奇数就放在结果的右边， 用两个指针来移动就可以。
当然也可以遍历两次， 但是那样就没有必要。

java题解代码

public int[] sortArrayByParity(int[] A) {
    int[] ret = new int[A.length];
    int left = 0, right = A.length-1;
    for(int i = 0; i < A.length; ++i) {
        if(A[i]%2 == 0) {
            ret[left++] = A[i];
        }
        else {
            ret[right--] = A[i];
        }
    }
    return ret;
}

  • go语言笔记2:
    初始化多个变量的值要这样

var left, right int = 0, len(A)-1

而写成下面这样是不能通过编译的

var left=0, right=len(A)-1
  • go语言笔记3: ++操作不能直接用在数组取index的操作里面
ret[left++] = A[i]

func sortArrayByParity(A []int) []int {
    var ret = make([]int, len(A))
    // var left = 0, right = len(A)-1
    var left, right int = 0, len(A)-1
    for i := 0; i < len(A); i++ {
        if A[i]%2 == 0 {
            ret[left] = A[i]
            left++
        } else {
            ret[right] = A[i]
            right--
        }
    }
    return ret;
}

时间复杂度

O(N)

空间复杂度

O(1)
