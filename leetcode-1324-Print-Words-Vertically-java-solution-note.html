题目 leetcode 1324 Print Words Vertically

输入一个字符串，把里面的单词按竖的方向写，然后再把每一行的字符串打印出来.
然后要求每个输出的字符串没有Trailing spaces

比如

Input: s = “TO BE OR NOT TO BE”
Output: [“TBONTB”,“OEROOE”," T"]
Explanation: Trailing spaces is not allowed.
“TBONTB”
“OEROOE”
" T"

解题思路分析

  1. 首先把字符串分割成单词
  2. 然后第一次取单词的第一个字母
  3. 如果遇到单词的长度不够，那就用空格补齐
  4. 最后把trailing space取掉
  5. 输出的字符串的个数应该是原来的字符串里面长度最长的字符串的长度

java题解代码

public List<String> printVertically(String s) {
    String[] words = s.split(" ");
    int maxlen = 0;
    for(String word : words) {
        maxlen = Math.max(maxlen, word.length());
    }
    List<String> list = new ArrayList<>();
    for(int i = 0; i < maxlen; ++i) {
        StringBuilder sb = new StringBuilder();
        for(String word: words) {
            if(word.length() > i) {
                sb.append(word.charAt(i));
            }
            else {
                sb.append(' ');
            }
        }
        String item = sb.toString();
        list.add(item.replaceAll("\\s+$", ""));
    }
    return list;
}
