leetcode 1008 Construct Binary Search Tree from Preorder Traversal题java解题笔记

09 March 2019

题目 leetcode 1008 Construct Binary Search Tree from Preorder Traversal

  1. 定义二叉搜索树是一个没有重复元素的二叉搜索树
    a binary search tree is a binary tree where for every node, any descendant of node.left has a value < node.val, and any descendant of node.right has a value > node.val
  2. 然后输入一个先序遍历的结果
    a preorder traversal displays the value of the node first, then traverses node.left, then traverses node.right
  3. 要求返回树的完整结构

解题思路分析

  1. 这个做法递归比较直观， 从根节点开始找到比他大的数就是右边子树的开始， 然后对左子树和右子树分别再构造树就好
  2. 注意右子树是空的情况

java题解代码

public TreeNode bstFromPreorder(int[] preorder) {
    return tree(preorder, 0, preorder.length-1);
}

public TreeNode tree(int[] preorder, int from, int to) {
    if(from > to) return null;
    TreeNode root = new TreeNode(preorder[from]);
    boolean found = false;
    for(int i = from+1; i <= to; ++i) {
        if(preorder[i] > preorder[from]) {
            root.left = tree(preorder, from+1, i-1);
            root.right = tree(preorder, i, to);
            found = true;
            break;
        }
    }
    if(!found) {
        root.left = tree(preorder, from+1, to);
    }
    
    return root;
}

时间复杂度

平均O(N*lgN)， 最差O(N^2), 相当于对于树的每一层，都对N个元素做了一次访问， 树高平均是lgN, 最差是N

空间复杂度

O(N)
