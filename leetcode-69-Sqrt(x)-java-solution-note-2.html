手动控制精度的代码

这里感觉重点是要取right， 这样就可以避免向下取0的问题。
同时， 因为是double，已经迭代到一个比较高的精度了， 所以取了right， 然后向下取整， 那就没有问题

public int mySqrt(int x) {

    double left = 0.0, right = (double)x;
    while(Math.abs(left - right) > 0.00001) {
        double mid = left + (right-left)/2.0;
        System.out.println("left=" + left + ", right=" + right + ", mid=" + mid);
        long t = (long)(mid * mid);
        if(t == (long)x) {
            right = (int)mid;
            break;
        }
        else if(t > x) {
            right = mid;
        }
        else {
            left = mid;
        }
    }
    return (int)right;
}
