leetcode 122 Best Time to Buy and Sell Stock II解题笔记

09 June 2019

题目 leetcode 122 Best Time to Buy and Sell Stock II

  1. 输入一个数组，里面表示每天股票的价格， 可以做任意多次交易， 问最大可以获取多少利润。
    当然， 在一个时间只能hold一个股票， 就是买了以后必须卖出以后才可以再买入.

  2. 这个题目和前面一个题目的区别是前面是只能做一次交易， 现在可以做多次交易
    前面的解题报告在这里http://www.noteanddata.com/leetcode-121-Best-Time-to-Buy-and-Sell-Stock-java-solution-note.html

解题思路分析

  1. 从左向右扫描，
    a. 假设第一天的价格是buy， 然后如果遇到高的，就进行一次交易， 把利润加上. 同时更新buy为当前价格。
    这个对横跨多天的交易也是可以的， 因为(b-a) + (c-b) = c-a

b. 如果遇到价格低的, 就更新buy为当前价格

java题解代码

public int maxProfit(int[] prices) {
    if(prices.length <= 1) return 0;
    int max = 0;
    int buy = prices[0];
    for(int i = 1; i < prices.length; ++i) {
        if(prices[i] > buy) {
            max += prices[i]-buy;
            buy = prices[i];
        }
        else if(prices[i] < buy) {
            buy = prices[i];
        }
    }
    return max;
}

时间复杂度

O(N)

空间复杂度

O(1)

之前写的另外一个代码

以前的提交记录， 现在看之前的代码更简洁一些, 但是可能理解起来要稍微绕一点。
这里的思路是把任何相邻数据的正的差加上就可以了

public int maxProfit(int[] prices) {
    int max = 0;
    for(int i = 1; i < prices.length; ++i) {
        if(prices[i] > prices[i-1]) {
            max += prices[i] - prices[i-1];
        }
    }
    return max;
}
