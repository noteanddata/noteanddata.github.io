coursera的吴恩达机器学习课程第一周-cost function最简单的情况

  1. 课程视频: https://www.coursera.org/learn/machine-learning/lecture/N09c6/cost-function-intuition-i

  2. 在前面的linear regression里面， 定义了预测函数是 y = kx + b ，

  3. 定义预测的距离为欧式距离的平方, d = (y[x] - y)^2
  4. cost function可以设置成 H= sum(d) /2m

  5. 如果取最简单的情况， 设置b=0, y = kx, 其中， 实际的数据是[1,1], [2,2], [3,3],

  6. 那么， H其实是一个关于k的函数
  7. 再取最简单的情况，预测函数也设置成k = 1, 也就是y=x, 那么, H = 0, 也就是完美预测，距离是0，
  8. 然后尝试其他k的值，在二维坐标上把k和H的值画出来，其实是一个抛物线， 极点就是[1,0]
