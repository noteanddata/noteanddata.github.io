题目 leetcode 20 Valid Parentheses

输入一个字符串里面有’(’, ‘)’, ‘{’, ‘}’, ‘[’, ’]’, 判断这个字符串是否合法

解题思路分析

  1. 题目非常直接，用一个stack可以判断是否合法
  2. 记得pop的时候判断当前stack是否是空， 以及最后stack是否是空

java题解代码

public boolean isValid(String s) {
    Stack<Character> stack = new Stack<>();
    Map<Character, Character> map = new HashMap<>();
    map.put(')', '(');
    map.put(']', '[');
    map.put('}', '{');
    
    for(char ch: s.toCharArray()) {
        if(map.containsKey(ch)) {
            if(stack.isEmpty() || stack.pop() != map.get(ch)) {
                return false;
            }
        }
        else {
            stack.push(ch);
        }
    }
    return stack.isEmpty();
}

时间复杂度

O(N)

空间复杂度

O(N)

繁琐一点的代码

虽然题目很简单， 但是代码一样可以写的很繁琐

public boolean isValid(String s) {
    Stack<Character> stack = new Stack<>();
    for(char ch: s.toCharArray()) {
        if(ch == '(' || ch == '[' || ch == '{') {
            stack.push(ch);
        }
        else {
            if(stack.isEmpty()) {
                return false;
            }
            if(ch == ')') {
                if(stack.pop() != '(') {
                    return false;
                }
            }
            else if(ch == ']') {
                if(stack.pop() != '[') {
                    return false;
                }
            }
            else if(ch == '}') {
                if(stack.pop() != '{') {
                    return false;
                }
            }
        }
    }
    return stack.isEmpty();
}
