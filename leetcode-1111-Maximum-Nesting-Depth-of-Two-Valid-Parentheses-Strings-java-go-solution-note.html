题目 1111 Maximum Nesting Depth of Two Valid Parentheses Strings

这个题目意思比较难读
1. 首先输入是一个只有左括号和右括号的字符串
2. 然后定义合法的括号字符串就是左右可以匹配的， 就是可以用stack正确模拟的
3. 定义这个字符串的深度规则如下
depth(““) = 0
depth(A + B) = max(depth(A), depth(B)), where A and B are VPS’s
depth(”(” + A + “)”) = 1 + depth(A), where A is a VPS.
4. 然后从这个字符串里面取一个子序列，这个子序列也是合法的括号字符串， 然后剩下的子序列也是一个合法的括号字符串
并且使得max(depth(A), depth(B))最小
5. 返回这个分割的结果， 第一个子序列对应的是0， 第二子序列对应的是1

解题思路分析

  1. 首先求解最大深度， 然后这个分割要让max(depth(A), depth(B))最小的话, 那就是其中一个子序列的深度是max/2+max%2
  2. 那么， 遇到左括号的时候， 如果当前这个子序列的深度没有超过这个最大值，那就可以选；
    如果超过了， 那么左括号就不选， 等遇到右括号的时候会让深度下降， 那么后面又可以选左括号.
    遇到右括号的时候， 可以用贪心的思想，每次都先选， 这样可以保证一定会遇到结果

java题解代码

class Solution {
    public int[] maxDepthAfterSplit(String seq) {
        int max = 0;
        int count = 0;
        for(int i = 0; i < seq.length(); ++i) {
            if(seq.charAt(i) == '(') {
                count++;
                max = Math.max(max, count);
            }
            else {
                count--;
            }
        }
        
        int depth = max/2 + max%2;
        int[] ret = new int[seq.length()];
        count = 0;
        int acount = 0;
        boolean valid = true;
        for(int i = 0; i < seq.length(); ++i) {
            if(seq.charAt(i) == '(') {
                count++;
                if(count <= depth) {
                    acount++;
                    ret[i] = 0;
                }
                else {
                    ret[i] = 1;
                }
            }
            else {
                count--;
                if(acount > 0) {
                    acount--;
                    ret[i] = 0;
                }
                else {
                    ret[i] = 1;
                }
            }            
        }
        return ret;
    }
}

go语言题解代码

func maxDepthAfterSplit(seq string) []int {
    max, count := 0, 0
    for i := 0; i < len(seq); i++ {
        if seq[i] == '(' {
            count++
            if count > max {
                max = count
            }
        } else {
            count--
        }
    }
    
    depth := max/2 + max%2
    ret := make([]int, len(seq))
    count = 0
    acount := 0
    for i := 0; i < len(seq); i++ {
        if(seq[i] == '(') {
            count++
            if count <= depth {
                acount++
                ret[i] = 0
            } else {
                ret[i] = 1
            }
        } else {
            count--
            if acount > 0 {
                acount--
                ret[i] = 0
            } else {
                ret[i] = 1
            }
        }
    }
    return ret
}
