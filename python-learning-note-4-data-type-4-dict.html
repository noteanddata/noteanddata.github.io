python学习笔记4 – dict数据类型

dict就是其他语言里面的map, 不过python比较灵活，key的类型可以不一样， value的类型也可以不一样

mydict = {"brand": "apple", "year":2020}
>>> print(mydict)
{'brand': 'apple', 'year': 2020}

>>> print(mydict["brand"])
apple

>>> mydict["color"] = "blue"
>>> print(mydict)
{'brand': 'apple', 'year': 2020, 'color': 'blue'}

>>> mydict[1] = "great"
>>> print(mydict)
{'brand': 'apple', 'year': 2020, 'color': 'blue', 1: 'great'}
