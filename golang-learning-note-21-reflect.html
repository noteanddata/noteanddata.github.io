go语言的反射

05 September 2019

  1. go语言的反射功能主要在https://golang.org/pkg/reflect/这里实现
    阅读的时候的主要参考资料https://medium.com/capital-one-tech/learning-to-use-go-reflection-822a0aed74b7

  2. 主要功能有:
    a. 得到一个变量的类型
    reflect.TypeOf(i)
    返回的是一个reflect.Type, 里面会有很多信息
    https://golang.org/pkg/reflect/#Type

  • Kind(), 是struct, 还是slice等，
  • Field(i int), 可以得到字段的信息
  • Method(int) Method 可以得到方法的信息
    注意这里的方法是按照字母排序的, 而且是只有exported方法才在这里
    之前有人还为此感到困惑
    https://github.com/golang/go/issues/30688

b. 根据Type创建变量
newPtrVal := reflect.New(varType), passing in a reflect.Type

c. 得到变量的值
Finally, you can go back to a normal variable by calling the Interface() method.

d. reflect.MakeSlice和reflect.MakeMap来创建slice和map

intSlice := make([]int, 0)
	mapStringInt := make(map[string]int)

	// here are the reflect.Types
	sliceType := reflect.TypeOf(intSlice)
	mapType := reflect.TypeOf(mapStringInt)

	// and here are the new values that we are making
	intSliceReflect := reflect.MakeSlice(sliceType, 0, 0)
	mapReflect := reflect.MakeMap(mapType)

还有reflect.MakeFunc等函数
