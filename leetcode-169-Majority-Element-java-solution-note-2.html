题目 leetcode 169 Majority Element

输入一个长度是n的整数数组，里面有一个数出现的次数是>n/2的， 请找出这个数组

解题思路分析

之前写了用一个hashmap来统计次数的解题笔记， 这里写一个另外的方法， 可以直接对数组排序， 然后取nums[n/2]这个元素， 也就数中位数， 就是这个majority element.
这个可以直接分类判断，
- 如果排序后的这个数占用了[0, x]， 那么他必然出现在中位数n/2上， 因为x > n/2
- 如果排序后的这个数占用了[x, n-1]， 那么他必然出现在中位数n/2上， 因为必然存在x < n/2, 否则这个数就不是majority element;
- 除去上面两种情况外， 这个数还可能是从[x,y], 其中x > 0, y < n-1, 那么必然有 x < n/2 < y, 否则这个数必然也不符合majority element. 因为前面的关系如果不符合的话， 要么是 0 < x < y < n/2, 要么是 n/2 < x < y < n-1, 这两种情况下都存在y-x < n/2, 那么也不是majority element

java题解代码

public int majorityElement(int[] nums) {
    Arrays.sort(nums);
    return nums[nums.length/2];
}

时间复杂度

O(N*lgN)

空间复杂度

如果是算法本身，没有占用额外的空间，可以认为是O(1)； 但是快速排序的过程本身最少是O(lgN)的空间复杂度。
所以空间复杂度还是可以认为是O(lgN)
