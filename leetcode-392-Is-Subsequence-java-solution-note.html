Pinterest面试题 leetcode 392 Is Subsequence

输入两个字符串s和t，判断字符串s是否是t的Subsequence;

解题思路分析

  1. 这个题目可以brute force过， 但是清晰正确的理解这个brute force也是非常重要！
    假如 s = “abc”, t = “kahbgdc”
    a. 用两个指针i和j分别指向s和t的开头，如果对应的字符一样的话，那么两个指针就可以向前移动一步，因为问题就转化成
    s.substring(i+1)是否是t.substring(j+1)的subsequence了。
    因为如果s的指针不移动，那么问题就是s是否是t.substring(j+1)的subsequence了， 而这个时候如果成立的话， 那么s.substring(i+1)是否是t.substring(j+1)的subsequence也一定成立. 所以， 在这一步做贪心的选择一定是最优的。
    b. 如果对应字符不一样，那么肯定需要移动j

java题解报告

public boolean isSubsequence(String s, String t) {
    if(s.length() == 0) return true;
    for(int i = 0, j = 0; j < t.length();) {
        if(s.charAt(i) == t.charAt(j)) {
            i++;
            j++;
            if(i == s.length()) {
                return true;
            }
        }
        else {
            j++;
        }
    }
    return false;
}

时间复杂度

O(M+N) 相对于最多需要遍历两个字符串的长度

空间复杂度

O(1)

其他感想

  1. 这个题目还有dp的做法， binary search的做法，
  2. 还有followup， 就是固定t， 但是会有大量的s需要查询是否是t的subsequence, 如何能够提高速度？
  3. 我看到有人说如果是subsequence的话，大部分情况下可以用贪心？而substring的话，则很多时候需要dp？
    这个逻辑，一时还没有想清楚。 但是有一点，什么时候用贪心，什么时候用dp，为什么必须要dp，是应该好好总结搞清楚的，这样才可以应对大部分题目和新题。
    另外，还有一些情况下可以用二分，也是要好好理解的。
