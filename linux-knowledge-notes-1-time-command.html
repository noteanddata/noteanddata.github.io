Linux知识学习笔记1–time命令, user time, sys time

linux提供一个time命令， 可以打印一个进程的cpu时间， 比如对ls进行测量， 其中real时间表示实际物理上的时钟时间(clock time), user表示cpu在用户态的时间， sys表示cpu在内核态的时间

$ time ls

real	0m0.010s
user	0m0.003s
sys	  0m0.003s

所以, user+sys是总共的cpu时间， 这个时间不一定和real有直接的关系。有可能user+sys< real， 比如这个程序并没有花太多时间在cpu上。 也可能user+sys > real， 比如对于一个多核机器， 然后程序也是比较并行化的，可以充分发挥多核的优势
