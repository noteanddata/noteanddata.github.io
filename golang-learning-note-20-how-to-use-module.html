go语言如何使用module

04 September 2019

如何在go语言里面使用module，
https://github.com/golang/go/wiki/Modules

  1. Invoke the go command in a directory outside of the $GOPATH/src tree, with a valid go.mod file in the current directory or any parent of it and the environment variable GO111MODULE unset (or explicitly set to auto).
  2. Invoke the go command with GO111MODULE=on environment variable set.

比如下载了一个代码， 要本地编译的时候， 如果是按照$GOPATH/src目录的， 那么设置GO111MODULE=on； 如果是在$GOPATH/src目录以外的，那么需要在当前目录下有一个go.mod文件. 然后GO111MODULE没有设置或者设置成auto
