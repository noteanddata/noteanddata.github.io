苹果公司面试题 leetcode 258 Add Digits

输入一个非负整数， 把每一位的数字加起来， 然后不断循环这个过程，直到最后只有一位数字， 问这个最后的数字是多少？

Input: 38
Output: 2 
Explanation: The process is like: 3 + 8 = 11, 1 + 1 = 2. 
             Since 2 has only one digit, return it.

解题思路分析

  1. 试图从小样本总结规律
    1到9， 那肯定是直接返回
10-->1
11-->2
....
18-->9
19--> 10--> 1
20---> 2
21--> 3
27--> 9
28--> 10-> 1
...
36---> 9
37--> 10--> 1
  1. 从上面的数据可以看到，从1开始的数，结果就是从1到9的循环， 那么基本就是n%9, 当然，对于9，18，27这样可以被9整除的数，结果应该返回9而不是0

java题解代码

public int addDigits(int num) {
    if(num == 0) return 0;
    if(num%9 == 0) return 9;
    return num%9;
}

当然， 代码还可以更简洁一点， 可以直接这样

public int addDigits(int num) {
    return 1 + (num - 1) % 9;
}

时间复杂度

O(1)

空间复杂度

O(1)

brute force的代码

不断循环直到数字是个位数

public int addDigits(int num) {        
    while(num >= 10) {
        int v = 0;
        while(num != 0) {
            v += num%10;
            num /= 10;                
        }
        num = v;
    }
    return num;
}

下面是最brute force的代码，不断循环， 而且每次都转化成字符串，效率最低

public int addDigits(int num) {
    String s = "" + num;
    while(s.length() > 1) {
        num = 0;
        for(int i = 0; i < s.length(); ++i) {
            char ch = s.charAt(i);
            int v = (int)(ch - '0');
            num += v;
        }
        s = "" + num;
    }
    
    return Integer.valueOf(s);    
}
