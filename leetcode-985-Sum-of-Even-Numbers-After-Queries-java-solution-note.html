Leetcode 985 Sum of Even Numbers After Queries解题笔记

01 February 2019

题目 Leetcode 985 Sum of Even Numbers After Queries

  1. 输入一个整数数组A， 然后再输入一个int[][] queries, 其中queries[i][0]是一个值val,
    而queries[i][1]是一个index， 表示要在A[i]上加上前面的val
  2. 求对每一次queries操作以后， 数组A里面的所有偶数的和
  3. 每次操作都会永久的改变A[i]的值

解题思路分析

由于每次操作的结果都是恒定的， 可以先把数组里面的偶数的和先算出来， 然后针对每次操作，根据A[i]本来是否是偶数和val是否是偶数， 更新在sum上需要加减的数就好。

java题解代码

public int[] sumEvenAfterQueries(int[] A, int[][] queries) {
    int[] ret = new int[queries.length];
    int sum = 0;
    for(int v: A) {
        if(v%2 == 0) {
            sum += v;
        }
    }
    for(int i = 0; i < queries.length; ++i) {
        int index = queries[i][1];
        int val = queries[i][0];
        if(A[index]%2 == 0) {
            if(val%2 == 0) {
                sum += val;
            }
            else {
                sum -= A[index];
            }
        }
        else { // A[index]%2 == 0
            if(val%2 != 0) {
                sum += A[index] + val;
            }
            else { // do nothing for sum
                
            }
        }
        A[index] += val;
        ret[i] = sum;
    }
    return ret;
}

时间复杂度

O(M+N) , 其中N是数组A的长度， M是queries的次数

空间复杂度

O(M), 需要一个和queries同样长度的数组用来存储结果的大小
