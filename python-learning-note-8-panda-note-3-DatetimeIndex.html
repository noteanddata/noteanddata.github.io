python学习笔记8 – panda DatetimeIndex使用笔记

参考资料: https://pandas.pydata.org/docs/reference/api/pandas.DatetimeIndex.html

里面有很多很方便的日期函数, 比如直接取dayofweek

测试代码：

>>> import pandas as pd
>>> dates = pd.DatetimeIndex(['2010-01-01 01:15:00', '2010-01-01 01:30:00'])
>>> print(dates)
DatetimeIndex(['2010-01-01 01:15:00', '2010-01-01 01:30:00'], dtype='datetime64[ns]', freq=None)
>>> dates.year
Int64Index([2010, 2010], dtype='int64')
>>> dates.month
Int64Index([1, 1], dtype='int64')
>>> dates.day
Int64Index([1, 1], dtype='int64')
>>> dates
DatetimeIndex(['2010-01-01 01:15:00', '2010-01-01 01:30:00'], dtype='datetime64[ns]', freq=None)
>>> dates.dayofweek
Int64Index([4, 4], dtype='int64')
>>>
