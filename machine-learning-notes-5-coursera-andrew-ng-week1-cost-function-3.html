coursera的吴恩达机器学习课程第一周-cost function最简单的情况

  1. 课程视频: https://www.coursera.org/learn/machine-learning/lecture/nwpe2/cost-function-intuition-ii

  2. 在前面的视频里面， 对预测函数是 y = kx + b 做了最简单的情况， 就是设置b=0， 那么如果b不是0，会是什么情况呢？

  3. 那么， H就会是一个关于k,b两个变量的函数， 然后对很多[k,b]做计算， 会得到一个三维空间中的很多点， 形成一个曲面
  4. 用软件把这些点画出来，会发现这个三维曲面也是有一个极点，在这个极点上会得到最小的H值。这个曲面的形状类似一个向上三维的椭圆球

那么，这个极点的位置对应的k和b，就是我们要求的最优预测函数
