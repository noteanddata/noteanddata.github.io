17 November 2018

题目 Leetcode 938 Range Sum of BST

输入一个binary search tree的根节点， 以及两个数L和R， 要求返回在[L,R]范围的的数的和，

解题思路分析

  1. binary search tree， 就是左边所有的数都比这个节点小或者相等，右边所有的数都比这个节点大或者相等，并且每个子树都符合这个特点
  2. 如果根节点比R要大的话，那根节点肯定不符合要求，而右边的子树肯定也不符合要求，所以直接在左边递归就好
  3. 同理，如果根节点比L要小的话，那根节点和左边的子树也肯定不符合要求，所以直接在右边递归就好
  4. 其他情况，就说明根节点在L和R之间， 加上根节点然后在两个子树中递归就好

java代码

public int rangeSumBST(TreeNode root, int L, int R) {
    if(null == root) return 0;
    if(root.val > R) {
        return rangeSumBST(root.left, L, R);
    }
    else if(root.val < L) {
            return rangeSumBST(root.right, L, R);
    }    
    else {
        return root.val + rangeSumBST(root.left, L, R) + rangeSumBST(root.right, L, R);
    }
    
}

时间复杂度

每个节点只遍历一次， 所以时间复杂度是O(N)

空间复杂度

没有使用额外的空间， 所以空间复杂度是O(1)
