facebook面试题 leetcode 56 Merge Intervals

  1. 定义interval=[start, end]
  2. 输入interval列表， 然后把互相overlap的interval合并在一起
  3. 返回最后的合并好的interval的列表

解题思路分析

  1. 基本分析过程就是按start排序， 然后就可以把相邻的有overlap的interval合并， 取end最大的就是新的end
  2. 但是我看了花神的视频，发现这个过程有一个通用的算法名称解一类问题，叫做扫描线算法, 下次还得专门学习一下这个专题https://www.youtube.com/watch?v=6tLHjei-f0I

java题解代码

  1. 下面是我的代码， 我用了一个start/end表示当前的interval， 然后遇到一个新的时候生成一个interval对象
  2. 这个记得循环结束的时候
/**
 * Definition for an interval.
 * public class Interval {
 *     int start;
 *     int end;
 *     Interval() { start = 0; end = 0; }
 *     Interval(int s, int e) { start = s; end = e; }
 * }
 */
class Solution {
     public List<Interval> merge(List<Interval> intervals) {
        if(intervals.size() <= 1) {
            return intervals;
        }
        intervals.sort((Interval interval1, Interval interval2)-> interval1.start - interval2.start);

        List<Interval> resultList = new ArrayList<>();
        int start = intervals.get(0).start;
        int end = intervals.get(0).end;
        for(int i = 1; i < intervals.size(); ++i) {
            Interval interval = intervals.get(i);
            if(end >= interval.start) {
                end = Math.max(end, interval.end);
            }
            else {
                Interval item = new Interval(start, end);
                resultList.add(item);

                start = interval.start;
                end = interval.end;
            }

        }
        resultList.add(new Interval(start, end));

        return resultList;
    }
}

时间复杂度

O(N*lgN)

空间复杂度

O(N)
