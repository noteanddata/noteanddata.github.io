reactjs学习笔记7-在jsx里面返回多个html元素

在写reactjx代码的时候，有一段jsx是返回一个, 也就是返回表格里面的一行, 大概类似这样

list.map(item=>(
  <Tr>
    <Td>{item.name}</Td>
  </Tr>
  ))

然后我需要改成返回两行, 结果发现直接加上一个就报错， 原来JSX默认是只支持一个元素的,
一个解决方法是直接返回数组

const Pets = () => {
  return [
      <Dog />,
      <Cat />
  ]
}

还有一个解决办法是在外面加一个Fragment元素, 这个Fragment是不会被渲染成html的，相当于没有影响

const Pets = () => {
  return (
    <Fragment>
      <Dog />
      <Cat />
    </Fragment>
  )
}

jsx的底层实现

jsx的实现其实是React.createElement函数, 比如

<MyButton color="blue" shadowSize={2}>
  Click Me
</MyButton>

就是

React.createElement(
  MyButton,
  {color: 'blue', shadowSize: 2},
  'Click Me'
)
