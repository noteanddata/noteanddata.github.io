题目–leetcode 929. Unique Email Addresses

输入一串email地址， 其中@前面的部分做特殊处理， 如果是"+“， 那所有+后面的都忽略， 如果是”.“, 那么”."就去掉，
然后判断输出的email地址里面有多少个不同的email

Input: ["test.email+alex@leetcode.com","test.e.mail+bob.cathy@leetcode.com","testemail+david@lee.tcode.com"]
Output: 2
Explanation: "testemail@leetcode.com" and "testemail@lee.tcode.com" actually receive mails

解题分析

这个题目没有任何trick，就把@前面的部分对+和"." 分别做处理，然后放到set里面就好

代码

public int numUniqueEmails(String[] emails) {
  Set<String> set = new HashSet<>();
  for(String email: emails) {
      String[] arr = email.split("@");
      String local = arr[0];
      int index = local.indexOf("+");
      if(index >= 0) {
          local = local.substring(0, index);
      }
      local = local.replaceAll("\\.", "");
      set.add(local + "@" + arr[1]);
  }
  return set.size();
}

复杂度

时间复杂度

数组遍历一次， 每个字符串遍历2-3次(indexOf, substring, replaceAll), 所以复杂度是O(N*M), N是数组的长度， M是字符串的长度

空间复杂度

用了一个set， 所以空间复杂度是O(N)
