题目 leetcode 169 Majority Element

输入一个长度是n的整数数组，里面有一个数出现的次数是>n/2的， 请找出这个数组

解题思路分析

  1. 这个题目直观的做法就是建立一个hashmap， 然后遍历数组， 统计每个数出现的次数， 取出最大值，
    这样做时间复杂度是O(N), 空间复杂度也是O(N)
    本以为这是一个简单的题目， 可是看了讨论以后发现这个题目解法非常多， 有一些精妙的解法, 实在是叹为观止

java题解代码

实现小细节， 如果一件取到了majority的数， 那就直接break, 不能再返回

public int majorityElement(int[] nums) {
    Map<Integer, Integer> map = new HashMap<>();
    int max = nums[0];
    int maxcount = 1;
    for(int i = 0; i < nums.length; ++i) {
        int count = map.getOrDefault(nums[i], 0);
        map.put(nums[i], count+1);
        if(count+1 > maxcount) {
            maxcount = count+1;
            max = nums[i];
            if(count+1 > nums.length/2) {
                break;
            }
        }
    }
    return max;
}

一份错误的代码

判断majority的逻辑有bug， 应该是 count > nums.length/2

public int majorityElement(int[] nums) {
    Map<Integer, Integer> countMap = new HashMap<>();
    for(int i = 0; i < nums.length; ++i) {
        Integer count = countMap.get(nums[i]);
        if(count == null) {
            count = 0;
        }
        count++;

        if(count >= nums.length/2) {
            return nums[i];
        }

        countMap.put(nums[i], count);
    }
    return 0;
}
