leetcode 804 Unique Morse Code Words的java和go解题笔记

04 July 2019

题目 leetcode 804 Unique Morse Code Words

  1. morse编码是把英文字母映射到其他字符的一个方式， 每个字母都有一个固定的映射规则；
  2. 输入一个字符串数组， 问最后会有几个不同的morse编码

解题思路分析

思路非常直接， 搞一个set， 把每个字符串转换成morse编码，看看总共有多少个就好

java题解代码

public int uniqueMorseRepresentations(String[] words) {
    String[] table = new String[]{".-","-...","-.-.","-..",".","..-.","--.","....","..",".---","-.-",".-..","--","-.","---",".--.","--.-",".-.","...","-","..-","...-",".--","-..-","-.--","--.."};
    
    Set<String> set = new HashSet<>();
    for(String s : words) {
        StringBuilder sb = new StringBuilder();
        for(int i = 0; i < s.length(); ++i) {
            char ch = s.charAt(i);
            int index = (int)(ch - 'a');
            sb.append(table[index]);
        }
        set.add(sb.toString());
    }
    
    return set.size();
}

go题解代码

  1. go语言里面没有内置set， 只有map来代替
import "strings"
func uniqueMorseRepresentations(words []string) int {
    var retmap = make(map[string]bool)
    var codetable = [26]string{".-","-...","-.-.","-..",".","..-.","--.","....","..",".---","-.-",".-..","--","-.","---",".--.","--.-",".-.","...","-","..-","...-",".--","-..-","-.--","--.."}
    for i := 0; i < len(words); i++ {
        var word = words[i]
        var sb strings.Builder
        for j := 0; j < len(word); j++ {
            var ch = word[j]
            var index = ch - 'a'
            var code = codetable[index]
            sb.WriteString(code)
        }
        var code = sb.String()
        retmap[code] = true
    }
    return len(retmap)
}

时间复杂度

O(N*M) N是数组的长度, M是每个字符串的平均长度

空间复杂度

O(N)
