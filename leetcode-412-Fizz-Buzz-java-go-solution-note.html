leetcode 412 Fizz Buzz解题笔记

06 July 2019

题目 leetcode 412 Fizz Buzz

输入n， 输出n个字符串， 分别对应1到n的整数对应的映射
- 如果可以被3整除也可以被5整除, 那么返回FizzBuzz
- 如果可以被3整除, 那么返回Fizz
- 如果可以被5整除, 那么返回Buzz
- 否则返回i

java题解代码

class Solution {
    public List<String> fizzBuzz(int n) {
        List<String> list = new ArrayList<>();
        for(int i = 1; i <= n; ++i) {
            if(i%3 == 0 && i%5== 0) {
                list.add("FizzBuzz");
            }
            else if(i%3 == 0) {
                list.add("Fizz");
            }
            else if(i%5 == 0) {
                list.add("Buzz");
            }
            else {
                list.add(String.valueOf(i));
            }
        }
        return list;
    }
}

go题解代码

在go里面把数字转换成字符串可以用strconv.Itoa函数

import "strconv"
func fizzBuzz(n int) []string {
    var arr = make([]string, n)
    for i := 1; i <= n; i++ {
        if i%15 == 0 {
            arr[i-1] = "FizzBuzz"
        } else if i%5 == 0 {
            arr[i-1] = "Buzz"
        } else if i%3 == 0 {
            arr[i-1] = "Fizz"
        } else {
            arr[i-1] = strconv.Itoa(i)
        }
    }
    return arr
}
