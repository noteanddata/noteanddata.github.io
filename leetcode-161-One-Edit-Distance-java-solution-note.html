题目 leetcode 161 One Edit Distance

输入两个字符串，定义edit操作是添加，删除，或者替换字符， 问能否通过一次edit操作完成使得两个字符串变成一样

解题思路分析

  1. 这个题目可以直接沿用leetcode 72 Edit Distance的做法， 用动态规划求出距离， 然后判断距离是否是1
    http://www.noteanddata.com/leetcode-72-Edit-Distance-java-solution-rewrite.html
  2. 隐约感觉也许只要求是否是1的话，也许可以不用动态规划直接双指针？？

java题解代码

public boolean isOneEditDistance(String s, String t) {
    if(Math.abs(s.length() - t.length()) >= 2) return false;
    int[][] dp = new int[s.length()+1][t.length()+1];
    for(int i = 1; i <= s.length(); ++i) {
        dp[i][0] = i;
    }
    for(int j = 1; j <= t.length(); ++j) {
        dp[0][j] = j;
    }
    
    for(int i = 1; i <= s.length(); ++i) {
        for(int j = 1; j <= t.length(); ++j) {
            dp[i][j] = Math.min(dp[i-1][j], dp[i][j-1]) + 1;
            if(s.charAt(i-1) == t.charAt(j-1)) {
                dp[i][j] = Math.min(dp[i][j], dp[i-1][j-1]);
            }
            else {
                dp[i][j] = Math.min(dp[i][j], dp[i-1][j-1]+1);
            }
        }
    }
    return dp[s.length()][t.length()] == 1;
}

时间复杂度

O(M*N)

空间复杂度

O(M*N)
