什么是dummy variables

dummy variables,又叫做indicator variables, 中文翻译成哑变量, 指示变量或者虚拟变量，
表示一个变量的值只有两个值， 通常在统计分析里面直接对应成0和1

在python的pandas库里面又一个get_dummies函数会把类别变量转化成dummy变量
“Convert categorical variable into dummy/indicator variables.”

>>> import pandas as pd
>>> s = pd.Series(["bachelor", "master", "doctor"])
>>> print(s)
0    bachelor
1      master
2      doctor
dtype: object
>>> d = pd.get_dummies(s)
>>> d
   bachelor  doctor  master
0         1       0       0
1         0       0       1
2         0       1       0
