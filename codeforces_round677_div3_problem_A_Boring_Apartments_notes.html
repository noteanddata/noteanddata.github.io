codeforces round 677 div3 problem A - Boring Apartments解题笔记

题目大意

  1. 有一个人在叫号码，按照下面的顺序数数字, 也就是从每个数字1到9循环，在每个数字上每次重复这个数字到4位数，
1,11,111,1111 
2,22,222,2222
3,33,333,3333
........
9, 99,999,9999
  1. 然后会输入另外一个数字， 比如22， 那么就停止.这样当前的叫过的号码就是
    1,11,111,1111,2,22

然后把这些数字的位数加起来，这个例子里面就是1+2+3+4+1+2=13

解题思路

实现题，直接遍历循环就好。 不过在数字版本的实现的时候，还是错了两次！
并且， 虚拟比赛的时候总共花了15分钟, 读题目和实现还是太慢了。

java解题代码

import java.util.Scanner;

public class CF_Round_677_Div3_ProblemA_Boring_Apartments {
    public static void main(String[] args) {
        process();
    }

    public static void process() {
        Scanner scanner = new Scanner(System.in);
        int t = scanner.nextInt();
        for(int i = 0; i < t; ++i) {
            int n = scanner.nextInt();
            int ret = solve(n);
            System.out.println(ret);
        }
    }

    public static int solve(int n) {
        int ret = 0;
        for(int i = 1; i <= 9; ++i) {
            int v = 0;
            for(int j = 1; j <= 4; ++j) {
                v = v * 10 + i;
                ret += j;
                if(v == n) {
                    return ret;
                }
            }
        }

        return ret;
    }


}

版本1

import java.util.Scanner;

public class CodeForce_Round_677_Div3_ProblemA_BoringApartment {

    public static void main(String[] args) {
        process();
    }

    public static void process() {
        Scanner scanner = new Scanner(System.in);
        int t = scanner.nextInt();
        for(int i = 0; i < t; ++i) {
            int x = scanner.nextInt();
            int count = calculate(x);
            System.out.println(count);
        }
    }

    public static int calculate(int x) {
        int count = 0;
        String strx = String.valueOf(x);

        for(int i = 1; i <= 9; ++i) {
            StringBuilder sb = new StringBuilder();

            for(int j = 0; j < 4; ++j) {
                sb.append(i);

                count += sb.length();

                if(sb.toString().equals(strx)) {
                    return count;
                }
            }
        }
        return count;
    }
}
