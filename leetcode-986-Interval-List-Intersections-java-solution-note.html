leetcode 986 Interval List Intersections解题笔记

01 February 2019

题目 leetcode 986 Interval List Intersections

  1. 输入两个一维的interval列表， 其中每个列表里面的interval都是已经排序并且不相交的， 而且每个interval都是闭区间。
  2. 也就是说[a, b]表示所有满足a<=x<=b的x，
  3. 要求返回两个interval列表合并后相交的interval数组

解题思路分析

  1. 两个列表都已经排序， 那么直接可以使用双指针， 然后逐个往前走就好
  2. 每次先把不相交的直接跳过， 然后把符合要求的interval存下来， 然后继续向前走一个指针就好

java题解代码

/**
 * Definition for an interval.
 * public class Interval {
 *     int start;
 *     int end;
 *     Interval() { start = 0; end = 0; }
 *     Interval(int s, int e) { start = s; end = e; }
 * }
 */
class Solution {
    public Interval[] intervalIntersection(Interval[] A, Interval[] B) {
        int i = 0, j = 0;
        List<Interval> list = new ArrayList<>();
        while(i < A.length && j < B.length) {
            if(A[i].end < B[j].start) {
                i++;
            }    
            else if(B[j].end < A[i].start) {
                j++;
            }
            else {
                int start = Math.max(A[i].start, B[j].start);
                int end = Math.min(A[i].end, B[j].end);
                list.add(new Interval(start, end));
                if(A[i].end < B[j].end) {
                    i++;
                }
                else {
                    j++;
                }
            }
        }
        
        Interval[] arr = new Interval[list.size()];
        arr = list.toArray(arr);
        return arr;
    }
}

时间复杂度

O(M+N), 其中M和N分别是两个数组的长度

空间复杂度

O(Math.max(M,N))?, 其实取决于最后的结果的大小, 感觉最大会可能是和两个interval的个数一样？
