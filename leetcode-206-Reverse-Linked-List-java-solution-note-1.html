leetcode 206 Reverse Linked List解题笔记

02 June 2019

题目 leetcode 206 Reverse Linked List

反转一个链表

解题思路分析

  1. 这是一个简单题， 可以递归做，也可以不递归做
  2. 但是我今天第一次写居然写错了， 虽然在提交前认识到自己的错误， 但是没有一遍流写对，还是不应该，所以记录一下笔记
  3. 这里记录一下递归的代码
    a. 递归的边界条件判断
    b. 对第二个节点开始做递归
    c. 把第二个节点指向头节点， 注意这里要把head.next设置成null (bug就在这里 )

java题解代码

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        if(null == head || null == head.next) return head;
        ListNode next = head.next;
        ListNode nhead  = reverseList(head.next);
        next.next = head;
        head.next = null;
        return nhead;
    }
}

时间复杂度

O(N)

空间复杂度

之前认为这个空间复杂度是O(1), 实际上想了一下应该是O(N), 因为这个递归的深度就是N

错误的代码

忘记把head.next设置成null了，导致out of memory
