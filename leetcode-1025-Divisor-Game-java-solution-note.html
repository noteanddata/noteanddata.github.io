leetcode 1025 Divisor Game解题笔记

13 April 2019

题目leetcode 1025 Divisor Game

  1. 输入一个N， 然后有两个人轮流玩游戏， 轮到一个人的时候，需要选择一个x， 使得0 < x < N and N % x == 0.
  2. 然后把N换成N-x 让对手走
  3. 对某一轮，如果找不到一个x，那就输了

解题思路分析

  1. 暴力的做法就是直接循环递归做，但是这里有很多重复计算，所以可以用一个dp数组记录结果
  2. 这个题目比较诡异的是我用递归+记忆化表格做居然超时了， 可是数据量明明很小，不知道为什么

java题解代码

public boolean divisorGame(int n) {
    boolean[] dp = new boolean[n+1];
    for(int k = 1; k <= n; ++k) {
        for(int i = 1; i <= Math.sqrt(k) && i < k; ++i) {
            if(k%i == 0 && !dp[k-i]) {
                dp[k] = true;
                break;
            }
        }
    }
    return dp[n];
}

时间复杂度

O(N^2)

空间复杂度

O(N)
