Amazon面试题目 leetcode 62 Unique Paths

输入一个矩阵，有一个机器人从左上角出发，每次只能向右或者向下走一步，问总共有多少种不同的路径？

解题思路分析

  1. 这个就是动态规划来做， 每一个位置只有两种不同的来源，上面一个位置或者左边一个位置，那么只要知道上面和左边的走法有多少个就可以了。

java题解代码

public int uniquePaths(int m, int n) {
    // dp[i][j] = dp[i-1][j] + dp[i][j-1]
    int[][] dp = new int[m][n];
    dp[0][0] = 1;
    for(int i = 0; i < m; ++i) {
        for(int j= 0; j < n; ++j) {
            if(i >= 1) {
                dp[i][j] += dp[i-1][j];
            }
            if(j >= 1) {
                dp[i][j] += dp[i][j-1];
            }
        }
    }
    return dp[m-1][n-1];
}

时间复杂度

O(M*N)

空间复杂度

O(M*N)

一个写错的版本

忘记初始化dp[0][0]了， 这样始终都是0.
