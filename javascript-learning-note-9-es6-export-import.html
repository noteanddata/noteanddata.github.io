javascript学习笔记9–ES6中Module的export和import

  1. ES6里面的Module概念就是一个模块, 整个加载是在编译的时候决定的.
    在ES6之前，有一些AMD和CommonJS的社区解决方法， 有了ES6的Moudle, 之前的都不需要了
  2. export就是可以把一个模块里面的变量和函数导出，让其他模块可以使用
    比如
export var firstName = 'Michael';
export var lastName = 'Jackson';
export var year = 1958;

export function multiply(x, y) {
  return x * y;
};
  1. import就是把其他模块里面的变量和函数导入到当前模块中使用
    比如
import { firstName, lastName, year } from './profile.js';

其中, import的变量是只读的
