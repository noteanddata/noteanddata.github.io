机器学习练习笔记11-用sklearn做线性回归，预测小红书销量5-4, 尝试把lifecycle加入哑变量

其中， lifecycle的不是数值结果， 而是A, B, C, 这里可以用转化成哑变量的方式来做回归。
当然， 看到一些资料上其他人有把gender这些也转化成哑变量， 直觉上觉得应该没有必要，
感觉线性回归对于不同的数值还是可以直接回归到正确的参数的， 当然这只是我的一个感觉，没有严格的数学证明。
当然， 如果是一些类别变量， 感觉转化还是有意义的, 因为绝对数值本身没有意义, 具体可以参考这个
https://zhuanlan.zhihu.com/p/72065474

下面是运行结果， 感觉加入这个变量的效果可以忽略不计, 提升效果大约0.03%

代码

#!/usr/bin/env python
# coding: utf-8

import math
import pandas as pd
import numpy as np 
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.linear_model import LinearRegression
from statsmodels.formula.api import ols 

df = pd.read_csv("../mldata/little_red_book/little_red_book.csv")

print(df.describe())

print(df.corr()['revenue'].sort_values(ascending=False))


df = df[df['revenue'] < 5000]

x = pd.DataFrame(df['previous_order_amount'])

#collist = [' days_since_last_order ', '3rd_party_stores', 'lifecycle', 'engaged_last_30', 'age', 'gender']
collist = [' days_since_last_order ', '3rd_party_stores', 'engaged_last_30', 'age', 'gender']

y = pd.DataFrame(df['revenue'])

for col in collist:
    print("adding col=" + col)
    df[col] = df[col].fillna(df[col].mean())    
    x = x.join(df[col])
    model=LinearRegression()
    model.fit(x,y)
    score = model.score(x,y)
    print(score)


add_lifecycle = pd.get_dummies(df['lifecycle'], prefix='lifecycle')    
print(add_lifecycle)

print("adding lifecycle dummy ")
x = x.join(add_lifecycle)

model=LinearRegression()
model.fit(x,y)
score = model.score(x,y)
print(score)
