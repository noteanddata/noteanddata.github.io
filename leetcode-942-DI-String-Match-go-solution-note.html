题目 leetcode 942 DI String Match

对于一个只包含D和I的字符串， 返回一个整数数组， 使得符合前面的条件
Return any permutation A of [0, 1, …, N] such that for all i = 0, …, N-1:

If S[i] == "I", then A[i] < A[i+1]
If S[i] == "D", then A[i] > A[i+1]

解题思路分析

  1. 这个题目dfs的解法比较暴力 http://www.noteanddata.com/leetcode-942-DI-String-Match-java-solution-note.html， 贪心的思路在这里 http://www.noteanddata.com/leetcode-942-DI-String-Match-java-solution-note-2.html
  2. 基本的思想是， 每次都选最大的或者最小的数， 只要满足当前的解是合法的， 后面无法是从k个数里面继续构造， 这k个数存在相对大小关系，也一样可以构造

go题解代码

func diStringMatch(S string) []int {
    var ret = make([]int, len(S)+1)
    left, right := 0, len(S)
    for i, v := range(S) {
        if v == 'I' {
            ret[i] = left
            left++
        } else {
            ret[i] = right
            right--
        }
    }
    ret[len(S)] = left
    return ret
}

时间复杂度

O(N)
