题目 leetcode 11 Container With Most Water

  1. 输入一个正整数数组， 每个位置代表一个砖块， 然后选择两个点， 组成一个容器， 问最大的容器可以容纳多少水？

解题思路分析

  1. 暴力的O(N^2)的代码当然可以做，但是没用
  2. 这个题目双指针的特点比较明显， 但是关键是如何证明双指针是正确的。 下面先说思路
    a. 首先选择i=0, j = height.lenght-1, 然后计算当前的容量
    b. 面积=宽度*min(高度1， 高度2)
    c. 这时候，因为宽度已经是最大值了， 其他选择宽度都更小， 所以， 这时候，我们需要考虑其他选择会导致高度变高的;
    而如果在两个指针中移动高度更高的那个指针的话，那么所有的可能里面，高度只可能小于等于当前选择的高度； 所以面积不可能更大。
    所以所有高度较低的指针对应的面积都不需要考虑了，
    d. 所以，这时候需要移动高度较低的指针.
    e. 而这个过程是可以循环论证的，所以， 一直移动高度较低的指针就好

java题解代码

public int maxArea(int[] height) {
    int sum = 0;
    for(int i = 0, j = height.length-1; i < j;) {
        sum = Math.max(sum, (j-i) * Math.min(height[i], height[j]));
        if(height[i] < height[j]) {
            i++;
        }
        else {
            j--;
        }
    }
    return sum;
}

时间复杂度

O(N)

空间复杂度

O(1)
