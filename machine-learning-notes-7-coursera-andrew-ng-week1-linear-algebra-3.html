课程资料：
https://www.coursera.org/learn/machine-learning/lecture/aQDta/matrix-vector-multiplication

主要笔记：
1. 如果第一个矩阵的列数和第二个矩阵的行数一样， 那么两个矩阵就可以相乘
2. result[i][j] = a[i][0] * b[0][j] + a[i][1] * b[1][j] + … + a[i][n-1] * b[n-1][j]
3. 对应到linear regression 上， 用矩阵表示各个不同参数的值，就比较容易表达和计算
