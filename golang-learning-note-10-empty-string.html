go语言学习笔记--空字符串

23 August 2019

问题1–go语言里面是否允许string是nil?

答案是不允许, 下面这段代码是会报错的

var s string = nil

cannot use nil as type string in assignment

在传统语言比如java里面， 所有对象都是指针， 都可以是null的。
而go语言里面的逻辑是，nil是表示没有初始化的对象，值未知， 而string没有初始化就是一个空字符串

参考资料
https://stackoverflow.com/questions/36001477/why-cant-a-string-be-nil-in-go/36006313

问题2–go语言里面如何判断字符串是空字符串

下面两种做法都是可以的

if len(s) > 0 { ... }

和

if s != "" { ... }

所以， 下面是一个测试程序

package main
import "fmt"

func main() {
    var s string = ""
    if s == "" {
        fmt.Println("s is empty")
    } else {
        fmt.Println("s is not empty")
    }
    if len(s) == 0 {
        fmt.Println("s is empty as well")
    } else {
        fmt.Println("s is not empty")
    }
}

运行结果是

s is empty
s is empty as well
