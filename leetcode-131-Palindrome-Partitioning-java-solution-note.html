题目 leetcode 131 Palindrome Partitioning

输入一个字符串，将其分割成每个子字符串都是回文， 返回所有都分割组合列表

Input: "aab"
Output:
[
  ["aa","b"],
  ["a","a","b"]
]

解题思路分析

  1. 又是经典的backtrack(回溯法)题目， 代码结构和全排列，还原ip地址等backtrack题目一摸一样，
    http://www.noteanddata.com/leetcode-46-Permutations-java-solution-note.html
    http://www.noteanddata.com/leetcode-93-Restore-IP-Addresses-java-solution-note.html

  2. 思路就是每次取前面的字符串(prefix), 判断是否是回文， 是的话递归， 递归到所有字符串都用完了，就是一个正确的分割。

java代码

    public List<List<String>> partition(String s) {
        List<List<String>> allList = new ArrayList<>();
        helper(new ArrayList<>(), s, allList);
        return allList;
    }
    
    public void helper(List<String> cur, String remain, List<List<String>> allList) {
        if(remain.length() == 0) {
            if(cur.size() > 0) {
                allList.add(new ArrayList<String>(cur));
            }
            return;
        }
        for(int i = 0; i < remain.length(); ++i) {
            if(isPalindrome(remain, 0, i)) {
                String part = remain.substring(0, i+1);
                cur.add(part);
                helper(cur, remain.substring(i+1), allList);
                cur.remove(cur.size()-1);
            }
        }
    }
    public boolean isPalindrome(String s, int i, int j) {
        while(i < j) {
            if(s.charAt(i) != s.charAt(j)) {
                return false;
            }
            i++;
            j--;
        }
        return true;
    }
