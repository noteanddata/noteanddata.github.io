leetcode 94 Binary Tree Inorder Traversal解题笔记

15 June 2019

题目 leetcode 94 Binary Tree Inorder Traversal

对一个二叉树做in order遍历

解题思路分析

  1. 这个题目递归解法是非常直接了， 按左-中-右遍历就好
  2. 当然循环迭代的做法也应该熟练掌握

java题解代码

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> list = new ArrayList<>();
        dfs(root, list);
        return list;
    }
    
    void dfs(TreeNode node, List<Integer> list) {
        if(null == node) return;
        dfs(node.left, list);
        list.add(node.val);
        dfs(node.right, list);
    }
}

时间复杂度

O(N)

空间复杂度

O(lgN)

java迭代题解代码

  1. 首先肯定是要用一个stack
  2. 每次遇到一个新节点的时候，需要循环把node.left压栈
  3. 每次弹出一个新节点的时候， 需要访问node.right, 然后继续第二步，看是否要访问所有左子树
  4. 这里， 如果只用stack，不使用一个cur变量，无法避免重复压栈的情况，
  5. 所以， 需要一个cur变量表示当前遇到的新节点，
    然后就可以循环执行压栈操作
while(null != cur) {
    stack.push(cur);
    cur = cur.left;
}

同理， 每次pop以后， cur = cur.right, 然后可以跳回前面的循环

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> list = new ArrayList<>();
        
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        while(null != cur || stack.size() > 0) {
            while(null != cur) {
                stack.push(cur);
                cur = cur.left;
            }
            cur = stack.pop();
            list.add(cur.val);
            cur = cur.right;
        }
        
        return list;
    }
}

时间复杂度

O(N)

空间复杂度

O(lgN)
