题目 Leetcode 1 Two Sum

https://leetcode.com/problems/two-sum/description/
1. 输入一个整数数组， 和一个target的数， 要求返回两个index，使得这两个index上的数等于这个target
2. 题目保证输入里面有且仅有一个解, exactly one solution, 这个非常重要。后续可以思考一下如果题目不保证exactly one solution,
应该怎么做？ 原来的几个做法是否还通用？

解题思路分析

  1. 还是这个第一题， 再之前的解题报告里面写了两种做法，一个是排序以后双指针， 一个是用hashmap的做法
  2. 所以今天看了下原来的hashmap的做法，先遍历一次建立hashmap从value->set 的map， 然后再遍历判断是否存在两个不一样的index，
    这里因为题目限制有且仅有一个解， 其实一次遍历就可以，直接在遍历的过程中把value->index加上， 然后每次看target-nums[i]是否存在就好

java解题代码

public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for(int i = 0; i < nums.length; ++i) {
        Integer index = map.get(target-nums[i]);
        if(null != index) {
            return new int[]{index, i};
        }
        map.put(nums[i], i);
    }
    return null;
}

时间复杂度

O(N), 数组只遍历了一次

空间复杂度

O(N), 用了一个map保存数组的值
