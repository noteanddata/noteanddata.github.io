题目 leetcode 155 Min Stack

设计一个min stack, 需要完成如下功能

  push(x) -- Push element x onto stack.
  pop() -- Removes the element on top of the stack.
  top() -- Get the top element.
  getMin() -- Retrieve the minimum element in the stack.

解题思路分析

  1. 这个题目和http://www.noteanddata.com/leetcode-716-Max-Stack-java-solution-note.html有点类似， 但是区别是那边要popmax, 这里只需要getMin()就好
  2. 如果对最小值不需要删除的话，那就比较简单，所有的添加和修改和原来的stack是一样的，只是需要保存每个时刻的最小值，
    而每个时刻的最小值只有在push和pop的适合会发生变化
  3. 这样，可以再维护一个minstack，每次push一个元素的时候，push主栈的最小值，这样每次pop的时候，也pop出来，这样，对于所有的push和pop都有合适的操作； 而计算当前的最小值， 只需要在minstack和新push的元素里面取最小值就可以了

java题解代码

class MinStack {
    
    private Stack<Integer> stack;
    private Stack<Integer> mins;

    /** initialize your data structure here. */
    public MinStack() {
        this.stack = new Stack<>();
        this.mins = new Stack<>();
    }
    
    public void push(int x) {
        stack.push(x);
        if(mins.isEmpty()) {
            mins.push(x);
        }
        else {
            int v = Math.min(mins.peek(), x);
            mins.push(v);
        }
    }
    
    public void pop() {
        stack.pop();
        mins.pop();
    }
    
    public int top() {
        return stack.peek();
    }
    
    public int getMin() {
        return mins.peek();
    }
}

/**
 * Your MinStack object will be instantiated and called as such:
 * MinStack obj = new MinStack();
 * obj.push(x);
 * obj.pop();
 * int param_3 = obj.top();
 * int param_4 = obj.getMin();
 */

时间复杂度

所有操作都是O(1)

空间复杂度

O(N)
