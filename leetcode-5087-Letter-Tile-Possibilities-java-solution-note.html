题目 leetcode 5087 Letter Tile Possibilities

输入一个字符串， 问使用这个字符串里面的任意字符组成多少个不同的字符串
输入的字符串长度<=7

解题思路分析

  1. 看了一下输入的字符串长度比较小， 决定直接使用排列的backtrack方法， 然后用一个set来去重
  2. 当然， 这个方法不是特别好， 看了下其他人是没有用set来做的， 这个后面再学习

java题解代码

public int numTilePossibilities(String tiles) {
    Set<String> all = new HashSet<>();
    for(int i = 0; i < tiles.length(); ++i) {
        backtrack(tiles.toCharArray(), i, "", all);    
    }
    
    return all.size()-1;
}

void backtrack(char[] tiles, int from, String cur, Set<String> all) {
    all.add(cur);
    
    for(int i = from; i < tiles.length; ++i) {
        swap(tiles, i, from);
        backtrack(tiles, from+1, cur + tiles[from], all);
        
        swap(tiles, i, from);
    }
}

void swap(char[] tiles, int i, int j) {
    char t = tiles[i];
    tiles[i] = tiles[j];
    tiles[j] = t;
}
