题目 leetcode 7 Reverse Integer

把一个整数reverse一下

解题思路分析

  1. 这里注意到输入有负数， 但是实际上可以用一个统一的逻辑取处理， 负数也没有问题
    这个可以看下面的例子
    -321%10 = -1
    -32%10 = -2
    -3%10 = -3

  2. 这里要求如果溢出32位的话就返回0；
    当然， 如果用字符串处理就不会有溢出的问题， 如果用整数，java就要用long
    不过go好像ret := 0然后做类型推断以后处理没有问题. 这个细节后面可以再学习一下

  3. 当然， 如果转化成字符串处理也是可以的， 不过那样效率就低了， 要经过来回的转换

java题解代码

public int reverse(int x) {
    long ret = 0;
    while(x != 0) {
        ret = ret * 10 + x%10;
        x /= 10;
    }
    if(ret > Integer.MAX_VALUE || ret < Integer.MIN_VALUE) {
        return 0;
    }
    return (int)ret;
}

go题解代码

import "math"
func reverse(x int) int {
    ret := 0
    for ; x != 0; x = x/10 {
        ret = ret * 10 + x%10
    }
    if ret < math.MinInt32 || ret > math.MaxInt32 {
        return 0
    }
    return ret
}

go语言的字符串解法

这里主要是练习一下go的语法
转换成字符串要处理的问题比较多， 比如负数

import "strconv"
import "strings"
//import "fmt"
func reverse(x int) int {
    if x < 0 {
        return -reverse(-x)
    }
    s := strconv.Itoa(x)
    
    //fmt.Println(s)
    
    var sb strings.Builder
    for i := len(s)-1; i >= 0; i-- {
        sb.WriteByte(s[i])
    }
    
    //fmt.Println(sb.String())
    
    ret, _ := strconv.Atoi(sb.String())
    if ret < math.MinInt32 || ret > math.MaxInt32 {
        return 0
    }
    return ret
}
