05 April 2019

Google面试题 Leetcode 731 My Calendar II

  1. 设计一个calendar类， 对于任意一个时间段，允许一次冲突， 但是不能有两个冲突
    public boolean book(int start, int end)

解题思路分析

  1. 这个题目应该有比较高效的解法，想了一会儿没想出来， 结果看了下答案brute forc也可以通过， 所以这里先写一下brute force的代码，后面再学习高效的解法
  2. 基本思路是保存每个已经成功预定的interval, 然后也保存已经overlap的时间段， 然后每次book的时候，先和overlap的比较下有没有再次冲突，有再次冲突就return false； 否则就加入到已经成功预定的interval列表里面， 同时计算已经overlap的时间段

java题解代码

class MyCalendarTwo {
    
    static class Interval {
        private int start;
        private int end;
        public Interval(int start, int end) {
            this.start = start;
            this.end = end;
        }
    }
    
    private List<Interval> intervalList;
    private List<Interval> overlapList;

    public MyCalendarTwo() {
        this.intervalList = new ArrayList<>();
        this.overlapList = new ArrayList<>();
    }
    
    public boolean book(int start, int end) {
        for(Interval overlap: overlapList) {
            if(end <= overlap.start || start >= overlap.end) {
                continue;
            }
            return false;
        }
        
        for(Interval item: intervalList) {
            if(end <= item.start || start >= item.end) {
                continue;
            }
            overlapList.add(new Interval(Math.max(start, item.start), Math.min(end, item.end)));
        }
        intervalList.add(new Interval(start, end));
        return true;
    }
}

/**
 * Your MyCalendarTwo object will be instantiated and called as such:
 * MyCalendarTwo obj = new MyCalendarTwo();
 * boolean param_1 = obj.book(start,end);
 */

时间复杂度

每次操作都是O(N), 整体做N次操作的话(就是订N个时间段)，时间复杂度都是O(N^2)

空间复杂度

O(N)
