解题思路分析

  1. 这个转化过程有一个算法叫做Shunting Yard Algorithm(调度场算法)， 主要的思路是用一个堆栈。 这个算法是由Dijkstra发明的， 没错，就是那个最短路径算法里面的Dijkstra
    参考资料:
    https://web.stanford.edu/class/archive/cs/cs106b/cs106b.1126/lectures/05/Slides05.pdf第75页
  2. 算法过程
    首先准备一个堆栈用来保存操作符
    a. 如果遇到一个数，就直接输出到最终结果
    b. 如果遇到左括号，就压到堆栈
    c. 如果遇到右括号，那就把堆栈里面的操作符直接出栈到最终结果，直到遇到左括号
    d. 如果遇到操作符，
    –如果堆栈是空的，那么就直接把操作符压到堆栈
    –如果当前的操作符比堆栈顶部的操作符优先级要低或者一样， 这个时候， 前面所有的高优先级的操作符都可以先处理，循环从堆栈中弹出元素，直到当前元素的优先级比栈顶的元素高
    –如果当前的元素比栈顶的操作符优先级高，那么需要先把这个操作符压到堆栈里面（核心原理是， 如果遇到高优先级的操作符，那就先放到堆栈里面，直到遇到低优先级的操作符，才可以把前面的计算掉。 )

  3. 举例
    每次遇到一个操作符，是不知道自己是否能够直接计算的， 因为后面可能遇到更高优先级的操作符。
    比如3+4*5—> 3 4 5 * +

java题解代码

public List<String> convertToRPN(String[] expression) {
    // write your code here
    Stack<String> stack = new Stack<>();
    List<String> list = new ArrayList<>();
    for(String token: expression) {
        if(tryParseInt(token)) {
            list.add(token);
        }
        else {
            if(stack.isEmpty()) {
                stack.push(token);
            }
            else {
                if(token.equals("(")) {
                    stack.push(token);
                }
                else if(token.equals(")")){
                    while(!stack.isEmpty()) {
                        String top = stack.pop();
                        if(top.equals("(")) {
                            break;
                        }
                        list.add(top);
                    }
                }
                else {
                    
                    if(token.equals("+") || token.equals("-")) {
                        while(!stack.isEmpty()  && !stack.peek().equals("(")) {
                            list.add(stack.pop());
                        }
                    }
                    else if(token.equals("*") || token.equals("/")) {
                        while(!stack.isEmpty() && 
                                (stack.peek().equals("*") || stack.peek().equals("/"))) {
                            list.add(stack.pop());
                        }
                    }
                    stack.push(token);
                }
            }
        }
    }
    while(!stack.isEmpty()) {
        list.add(stack.pop());
    }
    return list;
}

boolean tryParseInt(String s) {
    try {
        Integer.parseInt(s);
        return true;
    }
    catch(Exception e) {
        return false;
    }
}

遇到加号和减号， 要对前面的乘号和除号做循环pop, 只pop一个是不够的

public List<String> convertToRPN(String[] expression) {
    // write your code here
    Stack<String> stack = new Stack<>();
    List<String> list = new ArrayList<>();
    for(String token: expression) {
        if(tryParseInt(token)) {
            list.add(token);
        }
        else {
            if(stack.isEmpty()) {
                stack.push(token);
            }
            else {
                if(token.equals("(")) {
                    stack.push(token);
                }
                else if(token.equals(")")){
                    while(!stack.isEmpty()) {
                        String top = stack.pop();
                        if(top.equals("(")) {
                            break;
                        }
                        list.add(top);
                    }
                }
                else {
                    String top = stack.peek();
                    if(top.equals("+") || top.equals("-")) {
                        if(token.equals("+") || token.equals("-")) {
                            list.add(stack.pop());    
                        }
                        stack.push(token);
                    }
                    else if(top.equals("*") || top.equals("/")) { // top == "*" or "/"
                        list.add(stack.pop());
                        stack.push(token);
                    }
                    else { // top == "("
                        stack.push(token);
                    }
                }
            }
        }
    }
    while(!stack.isEmpty()) {
        list.add(stack.pop());
    }
    return list;
}

        Integer.parseInt(s);
        return true;
    }
    catch(Exception e) {
        return false;
    }
}
