题目 leetcode 191 Number of 1 Bits

输入一个整数， 问里面有多少位是1

解题思路分析

两种办法，
1. 一种是直接循环32次，然后右移以后取最右边的一位
2. 第二种是每次做无符号右移， 然后取最右边一位， 直到数字不是0为止

这里主要需要考虑的是，数字可能是负数， 所以如果是第二种做法，需要

java题解代码

这里注意， 可能是负数

public int hammingWeight(int n) {
    int count = 0;
    while(n != 0) {
        count += (n & 1);
        n = n >>> 1;
    }
    return count;
}

时间复杂度

O(32)

空间复杂度

O(1)

java题解代码2

public int hammingWeight(int n) {
    int count = 0;
    for(int i = 0; i < 32; ++i) {
        int v = n >> i;
        if((v & 1) == 1) {
            count++;
        }
    }
    return count;
}

错误代码1

这里有两个bug， 第一个是没有做无符号右移

public int hammingWeight(int n) {

        count += (n & 1);
