Amazon面试题 Leetcode 136 Single Number

输入一个整数数组，其中有一个数只出现一次，其他的数都正好出现两次。 求这个只出现一次的数是哪个?

解题思路分析

  1. 一个直观的做法当然是建一个map统计一下，当然这个做法在现在的面试里面肯定已经没用了。
  2. 另外一个有趣的做法是对所有的数做异或， 最后剩下的数就是那个single number,
    为什么？ 因为任何一个数异或他自己就是0

java题解代码

public int singleNumber(int[] nums) {
    int result = 0;
    for(int i = 0; i < nums.length; ++i) {
        result = result ^ nums[i];
    }
    return result;
}

时间复杂度

O(N)

空间复杂度

O(1)
