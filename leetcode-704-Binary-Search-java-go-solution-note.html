题目 leetcode 704 Binary Search

在一个排序数组里面搜索一个target，返回这个index

解题思路分析

教科书的二分搜索， 用两个指针每次扔掉一半

java题解代码

感觉以前写的java代码还是略繁琐了一点

class Solution {
    public int search(int[] nums, int target) {
        int begin = 0, end = nums.length-1, middle = begin + (end-begin)/2;
        while(begin <= end) {
            if(nums[middle] == target) {
                return middle;
            }
            else if(nums[middle] > target) {
                end = middle-1;
            }
            else {
                begin = middle + 1;
            }
            middle = begin + (end-begin)/2;
        }
        return -1;
    }
}

go题解代码

有一个要点是循环的条件设置成left <= right, 这样所有的判断都可以在循环里面完成, 外面不需要额外的逻辑

func search(nums []int, target int) int {
    for left, right := 0, len(nums)-1; left <= right; {
        mid := left + (right-left)/2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid+1
        } else {
            right = mid-1
        }
        
    }
    return -1
}

时间复杂度

O(lg(N))

空间复杂度

O(1)
