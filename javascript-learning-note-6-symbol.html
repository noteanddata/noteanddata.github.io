javascript笔记6–ES6里面的Symbol

  1. ES6里面的Symbol, 是一种新的基本类型， 主要是解决属性重名的问题, javascript里面默认属性相当于字符串， 所以不同对象之间的同名属性在操作的时候会覆盖， Symbol可以每次生成一个全局唯一的值，可以用来做属性，

演示代码如下

function testSymbol1() {  
  let person1 = {name: "jack"}
  let person2 = {name: "rob"}
  var result = {}
  Object.assign(result, person1, person2)
  console.log(result) // { name: 'rob' }
}

function testSymbol2() {  
  let person1 = {[Symbol('name')]: "jack"}
  let person2 = {[Symbol('name')]: "rob"}
  var result = {}
  Object.assign(result, person1, person2)
  console.log(result) // { [Symbol(name)]: 'jack', [Symbol(name)]: 'rob' }
}
