go语言里面的nil的意思

https://stackoverflow.com/questions/35983118/what-does-nil-mean-in-golang
简单的说nil是表示zero, 也就是没有初始化的值， 可以用来表示pointers, interfaces, maps, slices, channels and function types

但是nil不能指向struct, 也不能指向int这些类型

代码1

var a int
a = nil
fmt.Println("a=", a)

上面的代码会报错

cannot use nil as type int in assignment

代码2

package main

import "fmt"

func main() {
	var item = test()
	fmt.Println("item=", item)
}

type cat struct {
}

func test() *cat {
	return nil
}

上面的代码可以编译通过

代码3

package main

import "fmt"

func main() {
	var item = test()
	fmt.Println("item=", item)
}

type cat struct {
}

func test() cat {
	return nil
}



cannot use nil as type cat in return argument
