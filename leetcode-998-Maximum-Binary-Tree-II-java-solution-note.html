leetcode 998 Maximum Binary Tree II解题笔记

23 February 2019

题目 leetcode 998 Maximum Binary Tree II

  1. 定义一个maximum tree: a tree where every node has a value greater than any other value in its subtree
    也就是每个树的根节点都比这个树的所有子节点要大
  2. 输入一个maximum tree的根节点和一个新的val， 要求把这个val加入到这个树，并且仍然保持这个树是maximum tree

解题思路分析

这个题目比较直观， 直接递归做就好， 如果val比根节点大， 那么让val成为新的根节点； 否则的话，就加入到任意一个子树

java题解代码

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode insertIntoMaxTree(TreeNode root, int val) {
        if(null == root) {
            return new TreeNode(val);
        }
        else {
            if(root.val < val) {
                TreeNode nroot = new TreeNode(val);
                nroot.left = root;
                return nroot;
            }
            else {
                root.right = insertIntoMaxTree(root.right, val);
                return root;
            }
        }
    }
}

时间复杂度

最多遍历树的高度， 所以是O(lgN)

空间复杂度

没有使用额外空间， 所以是O(1)
