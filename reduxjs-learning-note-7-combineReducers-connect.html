reduxjs学习笔记7–combineReducers和connect

这里记录一下combineReducers和connect两个方法的笔记， combineReducers是redux自己的方法， connect用在react-redux里面

combineReducers方法

combineReducers非常直观， 就是把多个reducer合并成一个reducer

import cp1 from '../components/cp1/reducer'
import cp2 from '../components/cp2/reducer'

const rootReducer = redux.combineReducers({
    cp1,
    cp2
});

const store = redux.createStore(rootReducer)

export default store;

connect方法

connect方法是从UI组件生成容器组件， 也就是UI组件是纯粹显示UI的, 容器是管理状态的， 然后通过connect把UI组件和容器组件连接起来,

import { connect } from 'react-redux'

const VisibleTodoList = connect(
  mapStateToProps,
  mapDispatchToProps
)(TodoList)

mapStateToProps参数是一个输入逻辑， 每次state发生变化， 这个函数都会被调用， 也就是订阅了state的变化，

mapDispatchToProps是一个输出逻辑, 定义了哪些用户的操作需要定义成action， 然后传递给props

const mapDispatchToProps = (
  dispatch,
  ownProps
) => {
  return {
    onClick: () => {
      dispatch({
        type: 'SET_VISIBILITY_FILTER',
        filter: ownProps.filter
      });
    }
  };
}
