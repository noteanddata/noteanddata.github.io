25 April 2019

Airbnb面试题目 leetcode 324 Wiggle Sort II

输入一个整数数组， 要求返回数组是nums[0] < nums[1] > nums[2] < nums[3]

Input: nums = [1, 5, 1, 1, 6, 4]
Output: One possible answer is [1, 4, 1, 5, 1, 6].

题目假设一定有一个答案

解题思路分析

  1. 这个题目和前面一个wiggle sort的区别是， 这里不能有相等的数， 所以这里主要是对于有几个连续相等的数不能直接按排序来交换了
    http://www.noteanddata.com/leetcode-280-Wiggle-Sort-java-solution-note.html

  2. 这里可以先对数组进行排序， 然后把数组从中间分成左右两部分，然后每次从每部分里面取一个， 可以保证
    a[0] < a[1], a[1] > a[2], a[2] < a[3]…..

由于可能有一些相等的数， 所以这个取必须每部分从后向前取， 从前向后取会出错, 比如[3,4,4,5]

java题解代码

public void wiggleSort(int[] nums) {
    if(nums.length == 1) return;
    int[] temp = Arrays.copyOf(nums, nums.length);
    Arrays.sort(temp);
    int smallIndex = (nums.length+1)/2-1;
    int largeIndex = nums.length-1;
    for(int i = 0; i < nums.length; ++i) {
        nums[i] = (i%2 == 0) ? temp[smallIndex--] : temp[largeIndex--];
    }
}

时间复杂度

O(N*lgN)

空间复杂度

O(N)
