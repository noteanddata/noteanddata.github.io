leetcode 811 Subdomain Visit Count解题笔记

17 June 2019

题目 leetcode 811 Subdomain Visit Count

  1. 输入是一个count–>domain的映射的数组， 表示每个域名被访问的次数
    [“9001 discuss.leetcode.com”]
  2. 然后对于每个域名， 各个级别的域名都会被会被同样计数

  3. 问， 所有各个域名的访问次数的情况
    [“9001 discuss.leetcode.com”, “9001 leetcode.com”, “9001 com”]

解题思路分析

  1. 这个题目非常简单直接， 每次遇到一个域名以后，直接对域名进行解析然后判断就好, 用一个map存储每个域名对应的访问次数

java题解代码

public List<String> subdomainVisits(String[] cpdomains) {
    if(null == cpdomains) return new ArrayList<>();
    Map<String, Integer> map = new HashMap<>();
    for(String s : cpdomains) {
        String[] arr = s.split(" ");
        int count = Integer.valueOf(arr[0]);
        String domain = arr[1];
        map.put(domain, map.getOrDefault(domain, 0) + count);
        int index = domain.indexOf(".");
        while(index > 0) {
            String subdomain = domain.substring(index+1);
            map.put(subdomain, map.getOrDefault(subdomain, 0) + count);
            index = domain.indexOf(".", index+1);
        }
    }
    List<String> list = new ArrayList<>();
    for(Map.Entry<String, Integer> entry: map.entrySet()) {
        list.add("" + entry.getValue() + " " + entry.getKey());
    }
    return list;
}
