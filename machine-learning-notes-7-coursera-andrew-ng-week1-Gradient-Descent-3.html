https://www.coursera.org/learn/machine-learning/lecture/kCvQc/gradient-descent-for-linear-regression

主要笔记：
1. 对于梯度下降法， 之前的课程里说了， 可以求的局部最小值，不一定可以得到全局最小值
2. 不过，幸好对于linear regression来说， 整个曲面的形状一定是一个碗的形状，也就是convex (凸)的， 所以局部最小值就是全局最小值
3. 同时，对于linear regression来说， 求最小值的过程其实可以通过线性代数的方式直接求解，而不一定需要通过梯度下降的方式不断迭代获得
4. 梯度下降的迭代方式每次其实就是求一个偏导数?
