18 June 2019

题目 leetcode 1089 Duplicate Zeros

输入一个整数数组， 对里面的每个0都对拷贝一份， 在原来的数组上做in place改动

解题思路分析

  1. 设置一个queue，遇到0的话放进去两次， 其他情况放进去一次
  2. 然后每次从queue里面取数据
  3. 当然， 这个做法的问题在于需要O(N)的空间， 当然这个题目也可以用O(1)的空间， 这个当时比赛的时候做了一下感觉有点麻烦， 后来看了下其实可以做， 但是需要仔细分析下状态。
    但是， 比赛前几名的人也基本都是用O(N)空间的， 感觉比赛的时候还是应该怎么快怎么来， 不应该浪费时间

java题解代码

public void duplicateZeros(int[] arr) {
    Queue<Integer> queue = new LinkedList<>();
    for(int i = 0; i < arr.length; ++i) {
        queue.add(arr[i]);
        if(arr[i] == 0) {
            queue.add(0);
        }
        arr[i] = queue.poll();
    }
}

时间复杂度

O(N)

空间复杂度

O(N)
