coursera的吴恩达机器学习课程第一周-cost function笔记

课程视频: https://www.coursera.org/learn/machine-learning/lecture/rkTp3/cost-function

在前面的linear regression里面， 定义了预测函数是 y = kx + b ，
然后输入一堆训练数据[x1, y1], [x2, y2]….,
然后要得到一个预测函数， 那么， 由于实际数据肯定不可能完全符合一个预测函数， 所以必然有偏差，
那么，如果定义两个不同的函数那个好那个不好呢？
需要有一个量化的方法来衡量两个预测函数，
这时候就定义一个cost function, 然后计算各个预测函数的cost function的值，
这里，预测的准不准用预测结果和实际结果的欧式距离的平方来定义 d = (y[x] - y)^2, 也就是预测值和实际值的距离的平方
然后定义cost function = sum(d) /2m, 其中sum(d)就是指所有的d的和， 然后m是指样本的个数， 相当于取一下平均值，
最后除以2是为了一些后续处理的方便，对于cost function本身没有任何影响
