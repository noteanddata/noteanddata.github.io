leetcode 671 Second Minimum Node In a Binary Tree解题笔记

29 June 2019

题目 leetcode 671 Second Minimum Node In a Binary Tree

输入一个二叉树， 其中这个二叉树的每个节点， 要么是没有子节点， 要么当前节点的值是两个子节点里面比较小的这个
求这个二叉树的第二小的节点的值

解题思路分析

  1. 显然， 根节点是整个二叉树的最小的节点
  2. 然后要求第二小的节点， 一路递归下去， 把min和min2传递下去， 然后分别对两个子树进行递归判断， 然后返回min2就好

java题解代码

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public int findSecondMinimumValue(TreeNode root) {
        if(null == root) return -1;
        int min = root.val;
        int min2 = Integer.MAX_VALUE;
        min2 = dfs(root, min, min2);
        
        return (min2 == Integer.MAX_VALUE)? -1: min2;
    }
    
    public int dfs(TreeNode root, int min, int min2) {
        if(null == root) return min2;
        if(root.val != min && root.val < min2) {
            min2 = root.val;
        }
        if(null != root.left) {
            min2 = dfs(root.left, min, min2);
        }
        if(null != root.right) {
            min2 = dfs(root.right, min, min2);
        }
        return min2;
    }
}

时间复杂度

O(N)

空间复杂度

O(lgN)
