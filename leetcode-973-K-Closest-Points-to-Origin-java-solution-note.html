Leetcode 973 K Closest Points to Origin解题笔记

11 January 2019

题目 Leetcode 973 K Closest Points to Origin

  1. 输入二维平面上的一些点的坐标， 以及一个正整数K
  2. 要求返回离原点最近的k个点的坐标

解题思路分析

  1. 求距离最近的k个值， 那么，建立一个k+1大小的heap， 然后按照距离倒排序， 也就是max heap就好了
  2. 每次遇到一个点就放倒heap里面，如果大于k了， 就弹出一个，也就是把距离最远的弹出去， 最后剩下的就是距离最近的k个点

java题解代码

public int[][] kClosest(int[][] points, int k) {
    // max heap by distance 
    PriorityQueue<int[]> pq = new PriorityQueue<>(k+1, new Comparator<int[]>() {
        public int compare(int[] a, int[] b) {
            return  (b[0]*b[0]+b[1]*b[1])-(a[0]*a[0] + a[1]*a[1]);
        }    
    });
    for(int[] point: points) {
        pq.add(point);
        if(pq.size() > k) {
            pq.poll();
        }
    }
    int[][] list = new int[pq.size()][2];
    int i = 0;
    while(pq.size() > 0) {
        list[i++] = pq.poll();
    }
    return list;
}

时间复杂度

对N个节点分别做heap的插入和删除操作， 每个操作复杂度是logK, 所以整体复杂度是O(N*lgK)

空间复杂度

建立了一个大小为K的heap， 所以空间复杂度是O(K)
