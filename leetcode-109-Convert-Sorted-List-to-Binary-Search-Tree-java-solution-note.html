微软面试题目 leetcode 109 Convert Sorted List to Binary Search Tree

09 April 2019

微软面试题目 leetcode 109 Convert Sorted List to Binary Search Tree

  1. 输入一个已经排序的整数list， 返回一个高度平衡的二叉搜索树

解题思路分析

  1. 输入数据已经是排序的，那么要返回一个高度平衡的二叉搜索数，就需要把根节点定义到链表的中间，然后把左边和右边分别递归构造子树。
  2. 为了获得中间节点，这里的做法是把链表拷贝到一个ArrayList里面，这样就可以每次直接用index来访问。 当然这个题目还有更好的做法，后面再学习。

java题解代码

public TreeNode sortedListToBST(ListNode head) {
  List<ListNode> list = new ArrayList<>();
  ListNode node = head;
  while(null != node) {
    list.add(node);
    node = node.next;
  }
  return convert(list,0, list.size()-1);
}

TreeNode convert(List<ListNode> list, int from, int to) {
  if(from > to) return null;
  int mid = from + (to-from)/2;
  TreeNode root = new TreeNode(list.get(mid).val);
  if(null != root) {
    root.left = convert(list, from, mid-1);
    root.right = convert(list, mid+1, to);
  }
  return root;
}

时间复杂度

所有节点都需要遍历一次，所以时间复杂度是O(N)

空间复杂度

拷贝了整个链表O(N)
