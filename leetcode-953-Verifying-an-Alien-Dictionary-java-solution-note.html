leetcode 953 Verifying an Alien Dictionary解题笔记

08 December 2018

题目leetcode 953 Verifying an Alien Dictionary

  1. 外星人语言和我们一样用小写字母，但是顺序和我们不一样
  2. 给定一个字母顺序order， 和一个words数组， 判断这些words数组是否是按照外星人的order排序的

解题思路分析

  1. 首先针对order建立一个map， 从char->index, 表示他们的大小
  2. 然后对words数组里面的每相邻的两个字符串比较
  3. 注意的是， 不仅小于要直接返回， 大于也要直接返回， 也就是只要不相等，都应该返回比较结果
  4. 另外，前面都一样，但是长度不一样的情况也需要处理

  5. 很可惜， 幸好sample数据里面test case比较多，否则就写错了好几次， 其中3和4都没有自己注意到

java代码

class Solution {
    public boolean isAlienSorted(String[] words, String order) {
        Map<Character, Integer> map = new HashMap<>();
        for(int i = 0; i < order.length(); ++i) {
            char ch = order.charAt(i);
            map.put(ch, i);
        }
        
        for(int i = 0; i < words.length-1; ++i) {
            if(!inorder(words[i], words[i+1], map)) {
                return false;
            }
        }
        return true;    
    }
    public boolean inorder(String w1, String w2, Map<Character, Integer> map) {
        for(int i = 0; i < w1.length() && i < w2.length(); ++i) {
            char ch1 = w1.charAt(i);
            char ch2 = w2.charAt(i);
            int idx1 = map.get(ch1);
            int idx2 = map.get(ch2);
            if(idx1 < idx2) {
                return true;
            }
            else if(idx1 > idx2) {
                return false;
            }
        }
        return w1.length() <= w2.length();
    }
}

时间复杂度

O(N*M), 其中N是words数组的长度， M是每个字符串的长度

空间复杂度

O(1), 对长度为26的order建立了一个order index, 复杂度可以理解成是常数
